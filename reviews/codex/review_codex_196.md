**① 分类问题清单**

**一、叙事节奏**

1. 中段“暗线方案”信息密度过高，连续多段都是“提出问题 → 谢长庚解释 → 顾惟新理解/震惊”的会议推演节奏，行动感偏弱。
2. 朱载昱出场后有较长战略宣讲，连续多段以“他说 / 他顿了顿 / 他放下图”推进，段落功能相似，容易显得像设定说明。
3. 后半段连续抛出“贵妃折子、北镇抚司密档、象牙海豚密语、京里急电”四个悬念，转折太密，单章高潮有叠加过度风险。

**二、人物音色**

1. STYLE.md 写的是“朱载机”，草稿中为“朱载昱”，人物名存在设定不一致，需确认是否改名。
2. 朱载昱台词里“升级做事的方法”“体系的壳/骨”“今后五十年帝国在远洋的话事权”等表达偏现代战略汇报腔，皇帝音色与时代感略脱节。
3. 谢长庚部分台词有时过于概括化、口号化，如“明线保当下，暗线保长远。两条线并行，但绝不交叉。”比他应有的工程师短句更像战略总结。

**三、AI痕迹 / 禁词**

1. 指定禁词中，未发现“深深地 / 不禁 / 油然而生 / 心潮澎湃 / 眸子 / 苦涩地笑了 / 嘴角微扬 / 蓦然 / 悄然 / 莫名地”的精确出现。
2. 近似风险词有“深深的洞”，虽不是“深深地”，但接近禁词肌理，建议避免。
3. 有若干常见网文/AI式情绪句：“舱内的温度仿佛骤降”“全身的血液几乎瞬间冻结”“像两条正在搏命的蛇”“第三条线刚刚露出毒牙”，氛围有效，但略显模板化。

**四、场景真实感**

1. 最大硬伤：最后“刚译出的电报”不应带有郑玲珑宫里的熏香。电报是接收端译写的纸，不会传递原纸气味，除非另有解释。
2. 补给转运写得偏抽象，“小艇夜间转运”“边缘节点”“基础设施”缺少吨位、煤袋、淡水桶、吊索、潮汐、灯号、码头人手等具体执行细节。
3. “里斯本买弹药”与“一个月后舰队必须抵达直布罗陀”存在地理/时序疑问：里斯本在直布罗陀以西，若是抵达直布罗陀前的补给链，需要交代清楚。

**② 综合评分：3.5 / 5**

主线推进清楚，暗线补给与郑玄机悬念有吸引力；但中段说明感偏重，人物台词偶有现代战略腔，最后电报香味属于明显真实性漏洞，建议优先修。
