# 第16章 Codex 二次校对

- 叙事节奏：PASS
  闯宫、压场、带走、抹痕四段推进清楚，情绪上先炸开再迅速压平，符合宫廷阴线需要的狠劲。

- 人物音色：PASS
  郑玄机一句句都像是在按规矩办事，实则句句都在断活路；谢长庚仍是从“谁把那一页先拿走”这种细节去理解危险，和前几章一脉相承。

- AI痕迹：PASS
  没有明显套话，紧张感主要靠动作和秩序变化来造，不靠空泛形容词。

- 场景真实感：PASS
  宫门道、失鞋白袜、出入簿房、换页动作都很具体，“抹痕”这件事写得尤其扎实。

- 综合评分：4/5
  本章完成了宫廷阴谋线第一次正面显影，也把“缺页/换页”母题成功并线，可进入定稿。
