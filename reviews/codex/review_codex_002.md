**校对报告**

**1. 叙事节奏**
1. 前半段连续多段都采用“发现细节→做出判断→旁人质疑/反应”的推进模板，且段落长度接近，读起来容易出现匀速感。尤其从入格物司、量主管路、指出第七节问题，到主事复核这一串，信息在推进，但节拍变化不够大。
2. 正堂陈述部分连续数段都是“谢长庚指图说理一句，旁人给一个反应，再继续说理”，结构过整齐，像逐条汇报，戏剧张力略被削平。
3. 文中“像……”式比喻密度偏高，且常落在句尾作收束，连续阅读时会形成相似句式回响，削弱段落之间的呼吸差。

**2. 人物音色**
1. 谢长庚整体音色是对的，短句、直给、技术判断明确，没有明显说成郑玄机或刘恩的情况。这一项没有严重串音。
2. 轻微问题在于白德明、老匠、部分官员的台词区分度不够，基本都落在“老资格、带点轻慢”的同一声部，配角之间容易互相覆盖。
3. 结尾宦官传话功能完成了，但话语质地较通用；如果此人后续有身份分量，目前这句还没建立出独特宫廷腔。

**3. AI痕迹**
1. 重点禁词核查结果：未见你特别要求排查的这些词汇出现：`深深地 / 不禁 / 油然而生 / 心潮澎湃 / 眸子 / 苦涩地笑了 / 嘴角微扬 / 蓦然 / 悄然 / 莫名地`。
2. 没有明显“情绪空转型”AI套话，整体比常见模型文干净得多。
3. 轻微风险不在禁词，而在修辞习惯：比喻句偏多，且多为“像……”收尾，久了会有一点“写得太顺”的人工润色感。

**4. 场景真实感**
1. 工业场景前半段质感很好，有声音、热气、机油、铆钉、砖缝黑油等具体抓手，不存在“某个大机器”式空泛描写。最大问题在后半段：一进正堂，现场真实感转弱，更多变成图纸与口头论证。
2. “砖缝黑油渗深，所以能判断是长期微漏”是成立的；但“宽度能看年头”“已漏三十年”这类判断跨得略快，证据链稍显跳跃，像是主角正确得过于顺滑。
3. “震频”“应力错位”“缓振余量”等概念能撑起专业感，但术语偏现代工程语言，如果后文没有持续用同一套技术语汇体系承接，读者可能会觉得这段专业得太整，少了一点匠作时代的手感。

**综合评分：4分 / 5分**

整体评价：这一章的可读性和工业质感都在水准线上，禁词控制也干净。主要短板不是“写差了”，而是“写得太稳”了：节奏过匀、说明段略长、部分配角声部偏同质。只看叙事质量，属于基础很扎实、但还可以再打磨出波峰波谷的一章。
