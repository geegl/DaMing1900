校对报告如下。

**一、叙事节奏**

1. 中段“算账”部分连续采用“第一条/第二条/第三条/第四条”的说明式结构，信息清楚，但小说感下降，接近会议汇报。建议后续改稿时穿插朱载昱反应、舱外动静或谢长庚内心压力，打断均匀节奏。

2. “他想起……”连续出现四次：  
   “他想起离京前夜……”  
   “他又想起苏伊士夜战……”  
   “他还想起今早栈桥上……”  
   “最后，他想起了……”  
   这是最明显的同句式连续堆叠，容易显出机械感。

3. 后半段动作推进较密，但多次使用“就在这时”“紧接着”“然后”推进危机，节奏有效，却略显影视分镜化，文字层次偏直线。

**二、人物音色**

1. 谢长庚整体符合“短句、直接、技术判断”的设定，但“不能停”之后的一大段战略陈述过于完整、顺滑，像预写好的作战参谋报告，不太像临场承压下的谢长庚。可保留逻辑，但应增加停顿、删去部分排比。

2. 朱载昱的压迫感较强，帝王口吻基本成立。但“你滑头”“这种事儿会越多”“京里那帮老东西”等表达偏现代口语，若此角色应更接近朱载机式高密度、克制问话，可再收紧。

3. 曹吉祥作为宦官，台词只负责报信，未出现明显错位；但“陛下，京师急电。第五封了。”缺少刘恩式/宦官式间接敬语。如果曹吉祥设定也偏宫廷内侍，音色略显普通传令兵。

**三、AI痕迹与禁词**

1. 指定检查词未发现：  
   “深深地 / 不禁 / 油然而生 / 心潮澎湃 / 眸子 / 苦涩地笑了 / 嘴角微扬 / 蓦然 / 悄然 / 莫名地”均未出现。

2. 仍有少量网文模板化表达：  
   “瞳孔一缩”“静得可怕”“不祥的预感已经攥住了他的喉咙”“像一道诅咒”偏常见套路，建议重点替换。

3. 结尾“黑暗现在正张开更大的口，等着整支舰队”意象有效，但略接近 AI 式抽象威胁收束。若能落回具体海面、火光、汽笛或军舰反应，会更稳。

**四、场景真实感**

1. 海战与舰队场景总体较具体，有海图、锚地、信号灯、蒸汽小艇、接舷、岸炮、补给点等细节，真实感合格。

2. 个别描述偏现代视觉化，例如“红黄两色光在黑暗中疯狂旋转”，容易让人联想到现代警灯。若时代技术设定允许，需要前文支撑；否则建议改为舰灯、号灯、汽笛、铜钟等更贴近舰船场景的警戒细节。

3. “十几艘战船”“中型，三桅”“远洋快船样式”等表述略泛。关键幽灵船作为高潮道具，可以补一两个更硬的细节，如船型、吃水、帆索状态、船壳伤痕、舱口气味等，增强压迫感。

**综合评分：4 / 5**

章节主冲突清晰，结尾钩子强，人物关系和战略压力都立住了。主要问题集中在中段说明感偏重、局部句式重复，以及少量模板化惊悚表达。总体可用，建议做一次节奏和语感层面的精修。
