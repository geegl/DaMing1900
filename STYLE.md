# 《大明1900》写作风格指南

> 番茄小说平台专用。每章写作前必须对照本文档检查。

---

## 一、章节基础规格

| 项目 | 标准 |
|------|------|
| 字数 | 普通章 3500-7000汉字；关键章 5000-8000汉字（均不含章节标题） |
| 章节标题 | 4-8字，不剧透但要有张力，例："炮台夜雨"、"图纸上的血痕" |
| 更新节奏（参考） | 每章独立完整，有开有合 |
| POV | 第三人称限制视角，紧贴谢长庚；特殊章节（郑玄机视角、刘恩视角）需在章首标注 |

---

## 二、标准章节结构

```
【标题】
  - 正文首行必须是章节标题
  - 格式：# 第X章 章名
  - 标题不计入字数统计

【开头钩子】150-250字
  - 立刻进入场景，不铺垫背景
  - 前三句必须有一个悬念、冲突或反常细节
  - 禁止用"话说"、"且说"等说书腔开头

【情节推进】2000-2500字
  - 推进主要情节，每500字有一个小的信息更新或情绪变化
  - 对话与叙述比例约4:6
  - 每隔800字左右安排一次场景或情绪切换，防止平铺

【本章高潮/转折】800-1000字
  - 本章最重要的事件或信息揭示
  - 情绪密度最高点
  - 可以是爽点（技术突破、战斗胜利）或悬念爆发（真相揭示、意外事件）

【结尾悬念钩】150-250字
  - 必须让读者产生"下一章是什么"的强烈欲望
  - 钩子类型（每章选一种）：
    ① 行动悬念：主角正要做某件事，戛然而止
    ② 信息悬念：出现一条意外信息，含义未明
    ③ 威胁悬念：危险正在逼近，主角尚未察觉
    ④ 情感悬念：人物关系出现意外转变
  - 禁止用"欲知后事如何，且听下回分解"
  - 禁止用"（本章完）"
  - 禁止用阶段总结、提纲式收束、后续计划说明
  - 禁止使用类似"第二幕第一阶段，到此结束""第二幕第二阶段，正式开始""明天还有很多事。一步一步来"这类元叙事句
```

## 三、对话规范

**谢长庚对话风格**：
- 短句为主，少用从句
- 直接回应问题，不绕弯子
- 技术描述时用类比："这根管子的问题，就像……"
- 示例：`"不对。压力到这里就散掉了，你看这个接头。"`

**朱载机对话风格**：
- 常用反问测试对方
- 少有废话，每句话都有信息量
- 面对谢长庚时偶尔放松，会说没头没尾的感慨
- 示例：`"朕问你，这个图纸有没有问题。"` / `"你是第一个看懂的。"`

**郑玄机对话风格**：
- 文白夹杂，有时半句文言、半句白话
- 句子里总有另一层意思
- 习惯最后加一个无害的附带条件，实则是关键
- 示例：`"谢大人之议，甚为精妙。只是……此事或可缓议，毕竟眼下各司事务繁重。"`

**刘恩对话风格**：
- 宫廷腔，"奴婢"自称
- 敬语极多，但信息量精准
- 用间接方式传递危险信号
- 示例：`"奴婢多嘴，只是近日司礼监接了些文书，谢大人若有空，或许可留意着些。"`

## 四、叙事节奏规则

- **每5章必有一个"爽点章"**：明确的技术突破、战斗胜利、或反派受挫
- **每3章必有一个"伏笔章"**：埋下新的悬念线索，不必立刻解释
- **每10章必有一个"情感章"**：聚焦人物关系，降低行动密度，让读者喘息
- **高潮章前必有一个"压力积累章"**：局面不断恶化，读者感受到山雨欲来

## 五、禁用词表

### A. AI写作痕迹词（一律禁止）
```
深深地 / 不禁 / 油然而生 / 心潮澎湃 / 内心深处
不由自主 / 思绪万千 / 感慨万千 / 百感交集
眸子 / 苦涩地笑了 / 嘴角微扬 / 眼眸
蓦然 / 悄然 / 莫名地 / 不知为何
充满了……的气息 / ……的身影 / 彻底地
令人窒息 / 一股暖流 / 说不清道不明
```

### B. 满清相关词（绝对禁止，见BIBLE.md详细清单）
```
康熙 / 乾隆 / 雍正 / 任何清代年号
辫子 / 发辫 / 旗人 / 八旗
奴才（宦官用"奴婢"，不用"奴才"）
旗袍 / 马褂（作正式服饰）
```

### C. 时代错误词
```
总统 / 共和国 / 革命党（此时尚无此词）
洋大人（大明人不会自我矮化）
火车（存在，但不要滥用，需符合时代）
```

### D. 过度书面/翻译腔
```
然而（可用，但不要每段都有）
此时此刻 / 与此同时（滥用会显得生硬）
……的他/她（如"疲惫的他转过身"，尽量避免）
```

## 六、场景描写规范

**工业场景（核心场景，需要质感）**：
- 先声音后视觉：读者先"听到"气机运转，再"看到"场景
- 必须有具体的技术细节：不要写"一台很大的机器"，要写"三丈高的双缸立式气机"
- 工程师的感知维度：温度、震动、气味，不只是视觉

**战斗场景**：
- 聚焦谢长庚的判断过程，不是全景上帝视角
- 每个战术决定都需要有技术依据
- 死亡场景要节制，不刻意渲染，但要有重量

**宫廷场景**：
- 礼制细节要准确（明代，不是清代）
- 权力博弈通过台词暗示，不要直白叙述"郑玄机在算计某某"
- 环境细节反映人物权力地位

**海战/远征场景（第二幕）**：
- 保持地理真实感，每场战役前交代地形
- 气机故障是常见叙事工具（真实且制造紧张感）
- 补给、燃煤、润滑油是持续的战略压力，不要忽视

## 七、章节内一致性检查清单

写完每章后自查：
- [ ] 谢长庚的年龄/知识状态是否与时间线匹配
- [ ] 是否出现了禁用词
- [ ] 是否出现了满清相关内容
- [ ] 对话风格是否与人物圣经一致
- [ ] 结尾钩子是否足够强（自评：如果我是读者，我会继续看吗）
- [ ] 本章是否有至少一个"信息更新"（读者得到了新信息）
- [ ] 本章是否有至少一个"情绪波动"（主角或读者情绪有变化）
