# 《大明1900》去AI味注入协议

## 🎯 核心目标

**让AI文变成"读者看不出是AI写的顶级架空历史"**

---

## 📊 AI味的本质

| 问题 | 本质 | 解决方案 |
|------|------|---------|
| **平滑滤镜** | AI喜欢干净、均衡、中性的描写 | 注入感官反差、时代噪声 |
| **完美句式** | 句子结构过于工整、逻辑过于清晰 | 允许破格句、方言、断句 |
| **缺乏毛边** | 没有人类写作的"瑕疵感" | 注入口语化、时代俚语、情绪化破格 |
| **文脉缺失** | 缺少晚清/晚明特有的文风 | 参考文本注入、文风对齐 |

---

## 🔧 去AI味三板斧

### 第一板斧：参考文本注入

**每章生成前，贴3段同题材人类名家片段**

#### 参考文本库（修正版 - 无满清元素）

| 风格 | 来源 | 适用场景 |
|------|------|---------|
| **明末笔记** | 《陶庵梦忆》《板桥杂记》 | 江南市井、文人生活、晚明社会 |
| **明代小说** | 《金瓶梅》《三言二拍》 | 市井对话、人物塑造、底层生活 |
| **南明史料** | 《南明史》《南明纪事》 | 南明政治、军事、社会细节 |
| **明代公文** | 明代奏折选、诏令 | 官场语言、皇权语言 |
| **近现代过渡** | 《二十年目睹之怪现状》⚠️需去满清元素 | 社会转型、洋务运动 |
| **当代架空** | 《大明1937》《新宋》 | 历史架空结构、工业叙事 |

#### 注入方法

```python
# 生成第N章前
reference_texts = [
    ("老残游记", "片段1：描写晚清官场"),
    ("鹿鼎记", "片段2：韦小宝市井对话"),
    ("雍正皇帝", "片段3：康熙批奏折")
]

prompt = f"""
参考以下三段人类名家文本的文风：

1. 《老残游记》：{reference_texts[0]}
2. 《鹿鼎记》：{reference_texts[1]}
3. 《雍正皇帝》：{reference_texts[2]}

现在请用类似的文风，撰写第{N}章：{chapter_outline}

注意：
- 保持晚清/晚明的文脉感
- 允许破格句、方言、断句
- 避免"微微""淡淡""隐约"等中性词
"""
```

### 第二板斧：毛边噪声注入

**允许5-8%的非标准句**

#### 毛边类型

| 类型 | 示例 | 效果 |
|------|------|------|
| **方言断句** | "他娘的这炉子，又不灵了" | 市井感 |
| **重复修辞** | "不够。还不够。要更多。更多。" | 压抑感 |
| **时代错位俚语** | "这叫洋务，懂吗？洋务！" | 历史感 |
| **情绪化破格** | "阿牛死了。死了。陈铁没有哭。" | 冲击力 |
| **故意拖拍** | "蒸汽升腾。慢慢的。慢慢的。像死人的呼吸。" | 节奏感 |

#### 注入规则

```json
{
  "毛边比例": "每章5-8%的句子允许非标准",
  "方言占比": "底层人物对话中方言不少于20%",
  "破格句式": "每章至少3处故意破格",
  "重复修辞": "关键时刻允许重复3-5次"
}
```

### 第三板斧：温度+Top-p动态

**前30章高温，后期降温但加破格指令**

#### 参数配置

| 阶段 | 温度 | Top-p | 指令 |
|------|------|-------|------|
| **第1-30章** | 0.85 | 0.9 | "允许方言、破格句，保持市井感" |
| **第31-100章** | 0.8 | 0.85 | "适度破格，平衡工整与毛边" |
| **第101-200章** | 0.75 | 0.8 | "降低温度，但每10章强制注入5%破格" |
| **第201-220章** | 0.7 | 0.75 | "收尾阶段，回归工整但保留关键毛边" |

## 🔍 后处理润色

**每10章人工/另一个模型做一次"去AI味润色pass"**

### 润色清单

| 检查项 | 标准线 | 处理方式 |
|--------|--------|---------|
| **"微微/淡淡/隐约"数量** | ≤3次/章 | 替换为具体感官描写 |
| **工整句式占比** | ≤70% | 注入破格句 |
| **方言/俚语占比** | ≥15%（对话中） | 增加时代噪声 |
| **感官反差数量** | ≥3处/章 | 补充强反差描写 |
| **视觉瑕疵** | 5-8%非标准句 | 允许破格 |

### 润色prompt

```
请对以下章节进行"去AI味润色"，只改文风，不动剧情：

{chapter_content}

润色要求：
1. 将"微微""淡淡""隐约"替换为具体感官描写
2. 增加晚清/晚明特有的文脉感（公文体、说书味、洋务新名词）
3. 允许5-8%的破格句（方言、断句、重复修辞）
4. 注入至少3处感官反差（脏/乱/血腥 vs 精致/华丽/庄严）
5. 保持剧情和人物不变，只改文风
```

### 检测AI味

```python
def detect_ai_flavor(chapter):
    score = 0

    # 1. 中性词密度
    neutral_words = ["微微", "淡淡", "隐约", "似乎", "仿佛"]
    neutral_count = sum(chapter.count(w) for w in neutral_words)
    if neutral_count > 3:
        score += 0.3

    # 2. 工整句式占比
    sentences = chapter.split('。')
    regular_count = sum(1 for s in sentences if is_regular(s))
    if regular_count / len(sentences) > 0.8:
        score += 0.3

    # 3. 缺乏方言/俚语
    if not has_dialect(chapter):
        score += 0.2

    # 4. 缺乏感官反差
    if not has_sensory_contrast(chapter):
        score += 0.2

    return score
```

## ⚠️ 注意事项

### 1. 不要过度润色

**毛边不是错别字**，过度润色会变成"整容脸"。

### 2. 保持一致性

**同一人物的方言、口癖必须全书一致**。

### 3. 人工审核关键章节

**第1章、第50章、第100章、第200章**必须人工审核情感深度和文风。
