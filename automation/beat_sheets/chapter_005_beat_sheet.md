## 二、核心冲突

### 主要冲突
1. **军人使命 vs 生存本能**：老鬼亲手炸毁自己守护了十年的战舰
2. **个人尊严 vs 国家败亡**：北洋水师覆灭，如何有尊严地死去
3. **肉体毁灭 vs 精神幸存**：断臂之痛，成为"活着的伤疤"

### 次要冲突
1. 与老伙计锅炉的感情（拟人化）
2. 与12岁陈铁的偶遇（cameo）
3. 与丁汝昌提督的最后命令

## 三、情感弧光

```
绝望中的坚定 → 亲手毁灭的痛苦 → 肉体毁灭的剧痛 → 幸存者麻木
     ↓                ↓                  ↓               ↓
  接受命令      锅炉爆炸的瞬间      断臂的生理细节      虚无感
```

**关键情感节拍**：
1. **第1节拍（500字）**：接到自沉命令，内心抗拒但表面服从
2. **第2节拍（1000字）**：准备炸药，与锅炉的"对话"，回忆往事
3. **第3节拍（1500字）**：陈铁cameo，12岁孩子在码头偷看
4. **第4节拍（1000字）**：点燃导火索，爆炸，断臂的生理细节
5. **第5节拍（500字）**：被救起，望着定远号沉没，成为伤疤

## 四、感官细节清单（强制≥12处/千字）

### 视觉反差（脏/乱/血腥 vs 庄严）
1. 定远号龙旗破烂，但依然飘扬（庄严+衰败）
2. 煤烟覆盖的军服 vs 整齐的海军勋章
3. 锅炉房的红光（地狱般）
4. 断臂处的白骨与蒸汽（血腥+工业）
5. 威海卫海面燃烧的油迹（火在海面烧）
6. 陈铁12岁的瘦小身影在煤堆中（幼小+残酷环境）

### 听觉（蒸汽噪声）
1. 锅炉减压的嘶鸣（像垂死巨兽）
2. 定远号舰炮的轰鸣（远处日军炮击）
3. 断臂时自己的惨叫（短促，然后麻木）
4. 陈铁的呼吸声（孩子在煤堆后的紧张喘息）

### 嗅觉/味觉
1. 煤烟的硫黄味
2. 机油的金属腥味
3. 断臂处血腥与铁锈混合
4. 陈铁身上的汗酸味（孩子的体味）

### 触觉
1. 生铁扳手的冰冷（与锅炉的热对比）
2. 蒸汽烫伤的剧痛（左臂瞬间汽化）
3. 断肢处骨头碎裂的触感

## 五、台词清单（天津卫方言+军人命令式）

### 老鬼的台词特征
- 短促、粗粝、天津卫方言（"他娘的"、"狗日的"、"嘛"）
- 军人命令式（"造。"、"点。"、"走。"）
- 单字成句，极少修饰

**关键台词**：
1. 接到命令时："造。丁军门要它沉，咱就让它沉。"
2. 对锅炉独白："十年了……今儿个，咱俩一块儿走。"
3. 发现陈铁："滚！这儿没你事儿！"
4. 断臂后："……呵。"（苦笑，单字）

### 丁汝昌（简短命令）
- "定远，不能降。"
- "你，去办。"

## 六、结构分解

### 起（500字）：命令
- 【POV: 老鬼 - 第1视角】
- 威海卫黄昏，定远号搁浅在刘公岛浅滩
- 丁汝昌提督召集各舰管带，下达自沉命令
- 老鬼作为锅炉长，被要求炸毁锅炉
- 内心抗拒但表面服从（"造"）

### 承（1000字）：准备
- 下到锅炉房，最后一次检查
- 与锅炉的"对话"（拟人化，十年感情）
- 回忆：这锅炉是北洋水师的心脏，如今要亲手毁掉
- 准备炸药，计算爆炸当量
- **视觉细节**：煤烟、红光、机油、生铁

### 转（1500字）：交叉
- 12岁陈铁在码头煤堆后偷看（cameo）
- 老鬼发现陈铁，怒斥"滚！"
- 陈铁不退，问："大爷，这铁船……要沉了？"
- 老鬼沉默，然后："看好了，小子。这，就是咱大明的铁棺材。"
- **情感转折**：从个人悲痛，转为传递给下一代的残酷教育
- 点燃导火索

### 合（1000字）：毁灭
- 爆炸的瞬间，第一人称生理细节
- 蒸汽和铁片切断左臂
- 剧痛→麻木→虚无
- 被冲击波掀飞，落入海中
- 被救起，躺在小船上，看着定远号缓缓沉没
- 最后一眼：龙旗入水，煤烟散尽

### 尾声（500字）：伤疤
- 醒来，发现左臂缺失
- 摸着断肢处的烫伤疤痕
- 陈铁在床边（被渔民救起时一起救的）
- 老鬼对陈铁说的第一句话："以后跟着我，我教你修铁疙瘩。"
- **钩子**：老鬼从此成为"活着的战争伤疤"
