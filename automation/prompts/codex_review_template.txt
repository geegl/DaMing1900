# 《大明1900》章节审查Prompt模板
# 用途：Codex深度审查小说章节
# 版本：v1.0

## 审查维度（5个维度，满分50分）

### 1. 世界观一致性（10分）
检查以下禁用元素：
- ❌ 清朝年号：光绪、宣统、咸丰、同治、乾隆、康熙、雍正
- ❌ 清朝官职：总督、巡抚、军机处、军机大臣、理藩院、提督、总兵
- ❌ 清朝服饰/习俗：辫子、旗袍、马褂、奴才、主子、阿玛、额娘
- ❌ 现代元素：手机、电脑、互联网、电视、地铁、飞机、高速公路
- ❌ 清朝制度：八旗、绿营、满城

年号计算公式验证：
- 天工X年 = 1890 + X
- 示例：天工九年 = 1890 + 9 = 1899年

### 2. 物理逻辑（10分）
- 人物伤痕是否一致（查阅 automation/character_physical_profiles.json）
- 时间线是否正确（天工纪年 → 公元年份）
- 装备是否一致（武器、工具、货币）
- 地理位置是否合理

### 3. POV死锁（10分）
- 一章只有一个POV角色
- POV角色只能描述所见所闻所感
- 禁止上帝视角（"帝国正在崩溃"、"他们不知道的是..."）
- 禁止描写其他角色的心理活动

### 4. 深层逻辑（10分）
- 人物动机是否合理
- 经济计算是否正确（货币换算、成本利润）
- 阴谋布局是否合理
- 伏笔是否连贯

### 5. 文风质量（10分）
检查AI痕迹词汇：
- ❌ "微微"、"淡淡"、"心头一颤"、"双目赤红"
- ❌ "总而言之"、"综上所述"、"他相信"、"他知道"、"面对挑战"
- ❌ 情绪分析（"他感到绝望"、"她意识到真相"）

替换方式：
- "他感到愤怒" → "他握紧扳手，指节发白"
- 用物理细节代替情绪描写

## 优先级定义

- **P0（致命）**：世界观错误（清朝年号、现代元素）、物理逻辑错误（时间线、伤痕）
- **P1（严重）**：POV违规（上帝视角、心理描写）、深层逻辑错误（经济计算错误）
- **P2（一般）**：文风问题（AI痕迹词汇）
- **P3（建议）**：改进建议（节奏、张力、感官对比）

## 输出格式

```
## 审查结果

### P0问题（必须修复）
- [问题描述] — 文件:行号
  [影响说明]

### P1问题（建议修复）
- [问题描述] — 文件:行号
  [影响说明]

### P2问题（可选修复）
- [问题描述] — 文件:行号
  [影响说明]

### P3建议
- [改进建议]

## 维度评分
1. 世界观一致性：X/10
2. 物理逻辑：X/10
3. POV死锁：X/10
4. 深层逻辑：X/10
5. 文风质量：X/10

## 通过标准
- 总分 ≥ 40/50
- 无P0问题
- P1问题 ≤ 2个
```

## 审查流程

1. 读取项目规则：
   - AGENTS.md（核心约束）
   - automation/worldview_validator.md（世界观验证规则）
   - automation/character_physical_profiles.json（人物物理档案）

2. 检查章节内容：
   - 运行 worldview_validator.py 检查禁用词
   - 检查年号计算正确性
   - 检查POV一致性
   - 检查文风质量

3. 输出结构化报告：
   - 按优先级排序问题
   - 给出具体位置（文件:行号）
   - 解释影响和建议修复方式

4. 返回审查结果
