# 时间锚点体系 v1.0

**目的**：防止多POV叙事中的时间线错乱

---

## 时间锚点规则

### 每章必须包含时间标识

**方式一：铸造厂广播**
```
铸造厂的广播响了：
"庚子年三月初八，辰时。今日配给：窝头两个，咸菜一份。"
```

**方式二：旧报纸日期**
```
陈铁拿起那张《北洋报》，头版印着：庚子年三月初八。
```

**方式三：官方文书**
```
告示上的日期是：大明庚子年三月初八。
```

## 时间线铁律

1. **每章必须有时间锚点**（除回忆章节）
2. **时间只能向前**（不能倒流，除非明确标注"回忆"）
3. **死人不能复活**（阿牛第3章死，第10章只能回忆）
4. **伤痕不能消失**（陈铁左腿疤痕永久存在）

---

## 检查机制

**生成章节时**：
1. 读取 state.json 的当前时间
2. 检查前3章的时间
3. 确保时间顺序合理
4. 在章节开头注入时间锚点

**质量检查时**：
```bash
python3 automation/scripts/time_anchor_checker.py --chapter 10
```
