## 6. 特定AI味模式检查

### 模式1：过度描写情感
```
原文检查: 情感通过动作和对话表现，无直接心理形容词堆砌
示例: 老鬼的愤怒通过"指节发白"、"喃喃咒骂"表现，非"他非常愤怒"
结果: ✅ 安全
```

### 模式2：过于完整的因果链
```
原文检查: 存在合理的叙事跳跃，非每一步都解释
示例: "老鬼心里一动，脸上却没表情"（内心到外表，无过渡解释）
结果: ✅ 安全
```

### 模式3：词汇重复（非修辞性）
```
原文检查: 核心词汇（煤烟、生铁、断肢）重复是主题需要，非词汇贫乏
同义词使用: 恰当（"狗日的"与"狗娘养的"交替）
结果: ✅ 安全
```

### 模式4：完美句式
```
原文检查: 存在大量不完整句式、口语短句
示例: "造。" / "嗯。" / "放着，别碰。"
结果: ✅ 安全
```
