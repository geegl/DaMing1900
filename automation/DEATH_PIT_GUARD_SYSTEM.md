# 防死穴系统 v3.0 使用指南

## 🎯 系统定位

**防止AI长篇小说六大死穴**：
1. ❌ 人物工具化/同质化综合征
2. ❌ 多线脱钩/咬合失灵
3. ❌ 章节注水/过场填充症
4. ❌ 分析腔/文体漂移
5. ❌ 后期偷懒/输出平淡化
6. ❌ 情感债务未偿/灵魂缺失

---

## 📁 系统架构

### 核心文件

| 文件 | 功能 | 防止什么 |
|------|------|---------|
| **character_voice_profiles.json** | 人物语言特征库 | 人物同质化 |
| **foreshadowing_ledger.json** | 伏笔账本 | 多线脱钩 |
| **death_pit_guard.py** | 六大死穴检测器 | 全部死穴 |

---

## 🛠️ 使用方法

### 生成章节后立即检测

```python
from death_pit_guard import DeathPitGuard

guard = DeathPitGuard()

# 1. 生成章节内容
chapter_content = generate_chapter_with_ai()

# 2. 执行全部六大死穴检测
results = guard.full_check(chapter_content, chapter_num=10)

# 3. 生成报告
report = guard.generate_report(results, chapter_num=10)
print(report)

# 4. 如果有严重问题，要求AI修改
if any(results.values()):
    print("❌ 发现死穴问题，需要修改")
    # 返回给AI修改
else:
    print("✅ 通过防死穴检测")
```

## 🔍 六大检测功能详解

### 1. 人物同质化检测

**检测内容**：
- ✅ 平均句长是否符合人物特征
- ✅ 是否使用禁用台词
- ✅ 口语习惯是否一致
- ✅ 语言风格是否独特

**示例**：
```
❌ 第10章：张廷远平均句长35.2字，预期15-25字，可能同质化
❌ 第10章：老鬼使用了禁用台词「我深刻地认识到历史的必然性」
```

---

### 2. 多线脱钩检测

**检测内容**：
- ✅ 伏笔是否超期未回收
- ✅ 伏笔是否有进展（每10章）
- ✅ 主线-支线是否咬合

**示例**：
```
❌ 第60章：伏笔TIANGONG_001应在第50章回收，现已超期10章
⚠️ 第50章：伏笔MYSTERY_001已埋下40章，但无进展记录
```

---

### 3. 章节注水检测

**检测内容**：
- ✅ 核心冲突推进
- ✅ 动作描写占比
- ✅ 对话占比（过高也是注水）

**示例**：
```
⚠️ 第15章：未检测到核心冲突推进，可能为过场填充
⚠️ 第20章：动作描写占比2.3%，可能过于静态（注水嫌疑）
```

---

### 4. 文体漂移检测

**检测内容**：
- ✅ 禁用分析腔词汇
- ✅ 小说模式强制锁定

**禁用词库**：
```
由此可见、从叙事角度、这体现了、可以看作、
综上所述、总而言之、不难看出、值得注意的是、
这表明了、这反映了、从某种意义上说、可以说、
显然、毫无疑问、不可否认、客观地说
```

**示例**：
```
❌ 第25章：检测到分析腔「由此可见」
   上下文：...由此可见，陈铁的选择体现了底层觉醒...
```

---

### 5. 后期疲劳检测

**检测内容**：
- ✅ 张力密度对比（前后期）
- ✅ 细节密度对比
- ✅ 高张力词汇使用频率

**示例**：
```
⚠️ 第150章：张力密度0.15%，前期平均0.32%，下降53.1%，可能为后期疲劳
⚠️ 第180章：细节密度1.2%，可能过于简略（后期疲劳）
```

---

### 6. 情感缺失检测

**检测内容**：
- ✅ 关键章节情感高潮
- ✅ 人物内心挣扎
- ✅ 人性深度描写

**示例**：
```
⚠️ 第95章：关键章节但未检测到情感高潮场景
⚠️ 第100章：检测节点章节但无内心挣扎描写，可能缺乏人性深度
```

## ⚠️ 注意事项

### 1. 检测时机

**每章生成后立即检测**，不要等到积累几十章才发现问题。

### 2. 修改优先级

| 优先级 | 死穴类型 |
|--------|---------|
| **P0（必须立即修复）** | 文体漂移、人物同质化 |
| **P1（严重问题）** | 多线脱钩、章节注水 |
| **P2（需要关注）** | 后期疲劳、情感缺失 |

### 3. 人工审核

**AI检测不是万能的**，关键章节（如第1章、第50章、第100章、第200章）需要人工审核情感深度。
