## 一、明朝元素（必须使用）

### 1. 官制系统

**必须使用**：
- 三司制：布政使（民政）、按察使（司法）、都指挥使（军事）
- 内阁：首辅、次辅、大学士
- 六部：吏户礼兵刑工
- 禁卫军：锦衣卫、东厂、西厂
- 宦官：司礼监掌印太监、御马监太监

**禁止使用**（清朝元素）：
- ❌ 总督、巡抚（清朝固定官职）
- ❌ 军机处、军机大臣
- ❌ 理藩院
- ❌ 提督、总兵（清朝武官）

---

### 2. 年号系统（必须死锁）

| 皇帝 | 庙号 | 年号 | 在位时间 | 纪年对应 |
|------|------|------|---------|---------|
| **朱靖镇** | 显宗 | 天工 | 1891-1909 | 天工元年=1891年，天工十九年=1909年 |
| **朱靖渊** | 威宗 | 洪威 | 1909-1920 | 洪威元年=1909年，洪威十二年=1920年 |
| **泰安帝** | 顺宗 | 泰安 | 1919-1921 | 泰安元年=1919年，泰安三年=1921年 |

**禁止使用**：
- ❌ 光绪、宣统、咸丰、同治等清朝年号
- ❌ 民国纪年

---

### 3. 社会制度

**必须使用**：
- 工籍烙印（左臂烙印，如"津铸丙字七十四号"）
- 贱籍制度（丐户、贱籍，处理"不洁"之事）
- 铁本位货币（铁币会生锈贬值）
- 连坐法、配给制、宵禁

**禁止使用**：
- ❌ 八旗制度
- ❌ 辫子、旗袍
- ❌ 奴才/主子称呼（清朝宫廷用语）

---

### 4. 时间计算公式

```
天工纪年 → 公元年份
天工X年 = 1891 + (X - 1) = 1890 + X

示例：
天工九年 = 1890 + 9 = 1899年
天工十九年 = 1890 + 19 = 1909年
```

## 三、自动化验证流程

### 每章生成后必须执行：

```python
# 伪代码
def validate_worldview(chapter_content):
    errors = []

    # 1. 检查禁用词
    for forbidden_word in forbidden_words_list:
        if forbidden_word in chapter_content:
            errors.append(f"发现禁用词: {forbidden_word}")

    # 2. 检查年号
    if "光绪" in chapter_content or "宣统" in chapter_content:
        errors.append("发现清朝年号，必须使用明朝年号")

    # 3. 检查官制
    if "总督" in chapter_content or "巡抚" in chapter_content:
        errors.append("发现清朝官职，必须使用明朝官制")

    # 4. 检查时间线一致性
    year_mentions = extract_year_mentions(chapter_content)
    for mention in year_mentions:
        if not validate_year_consistency(mention):
            errors.append(f"时间线不一致: {mention}")

    return errors
```

## 五、人工审核要点

### 审核时必须检查：

1. ✅ 年号是否正确（天工/洪威/泰安）
2. ✅ 官制是否正确（三司制，非总督/巡抚）
3. ✅ 社会制度是否正确（工籍、贱籍、铁币）
4. ✅ 时间计算是否正确（天工X年 = 1890 + X）
5. ✅ 无清朝元素（辫子、旗袍、奴才/主子）
6. ✅ 无现代元素（手机、地铁、公共公园）

## 六、错误示例与修正

### 错误示例1：年号错误

**错误**：
```
光绪二十四年（1898年）：采购回炉铁500根
```

**修正**：
```
天工八年（1898年）：采购回炉铁500根
```

**公式验证**：天工八年 = 1890 + 8 = 1898年 ✅

---

### 错误示例2：官职错误

**错误**：
```
天津卫总督王将军
```

**修正**：
```
天津卫都指挥使王将军
```

**原因**：总督是清朝官职，都指挥使是明朝官职。
