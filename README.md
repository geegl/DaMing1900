## 世界观特色

### 核心设定
- **铁本位货币**：铁币会生锈贬值，月贬值率10%
- **工籍烙印**：底层工人左臂烙印，如"津铸丙字七十四号"
- **贱籍制度**：比工籍更底层的群体，专门处理"不洁"之事
- **技术代差**：大明落后西方8-20年
- **三司制**：布政使、按察使、都指挥使（明朝官制，非清朝）

### 阶级压迫
- **嗅觉标签**：劣质除锈剂的恶臭（持续7-10天）
- **视觉标签**：工籍烙印、铁币生锈
- **听觉标签**：机器轰鸣导致的听力下降
- **制度压迫**：连坐法、配给制、宵禁

## 写作原则

### 必须遵守
1. **每章≥3000字**
2. **POV死锁**：前30章只有陈铁/老鬼视角
3. **明朝官制**：禁用总督/巡抚等清朝术语
4. **技术边界**：1900年技术，无机甲/外骨骼
5. **冷硬风格**：用动作代替心理，禁用"微微/淡淡/心头一颤"

### 参考风格
- 《陶庵梦忆》：晚明市井生活
- 《金瓶梅》：底层真实感
- 《高堡奇人》：压抑感的日常化
- 《三体》：公理体系驱动叙事
