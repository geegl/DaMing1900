## 🎯 项目概述

**世界观**：大明延续至1900年，东方早期重工业化时代

**核心设定**：
- **时间**：1900年庚子年
- **背景**：八国联军逼宫、底层机工觉醒
- **科技**：蒸汽机、铁甲舰、电报、差分机
- **核心矛盾**：皇权 + 买办 vs 机工同盟

**三部曲结构**：
- **第一部：天工纪**（1-70章）- 潜龙勿用 - 底层挣扎
- **第二部：洪威纪**（71-165章）- 飞龙在天 - 暴兵反击
- **第三部：泰安纪**（166-220章）- 亢龙有悔 - 赤色觉醒

## 🔍 世界观亮点

### 历史分歧点

**1644年**：崇祯帝煤山自缢，太子朱慈烺南渡，建立南明

**256年后**：大明延续至1900年，完成早期工业化

### 核心矛盾

- **经济**：铁本位货币，买办垄断铁路煤炭
- **文化**：王阳明心学异化为"知铁合一"
- **阶级**：皇权 + 买办 vs 机工同盟

### 科技体系

- ✅ 蒸汽机、铁甲舰、电报、差分机
- ❌ 手机、地铁、公共公园、现代概念

## 📖 创作理念

### 学习借鉴

- **《高堡奇人》**：多线网状叙事、世界观传递方式、哲学沉思
- **《三体》**：帝国热力学定律、硬科幻逻辑
- **《陶庵梦忆》**：晚明市井生活、明代文风

### 核心原则

1. **具体场景**：不是概括，是画面
2. **人物性格**：通过对话展现，不是描述
3. **语言差异**：方言vs官话的阶层差异
4. **时代准确**：1900年大明，不是现代穿越
