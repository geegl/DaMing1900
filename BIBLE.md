# 《大明1900》世界观与人物圣经

## 一、基础纪年与时代背景

- **当前时间线**：大明王朝延续至今，从未被满清取代
- **年号体系**：第一皇帝朱载机用"崇工"年号（约1895-1908）；第二皇帝朱载昱用"宣征"年号（1908起）
- **故事主线时段**：崇工五年（1900年）开始，至宣征十三年（1921年）结束
- **都城**：北京（明代称"京师"）；南京为陪都，商业与财政中心
- **重要地理**：兰芳省省会古晋（婆罗洲西北岸）；哥打基纳巴卢为东北军事要塞港；巴达维亚（爪哇岛）为大明南洋贸易飞地，不属兰芳省

## 二、技术约束（格物工学体系）

### 存在的技术（1900年代）
- **气机**（蒸汽机）：成熟技术，高压型，核心动力来源；比西方蒸汽机铸造精度更高，但热效率理论不完整
- **铁甲舰**：大明主力海军装备，气机驱动，融合榫卯结构美学
- **电报**：已普及至省级；紫禁城内设总电报机房，由内廷掌控
- **煤油照明**：城市与工厂标配，替代鲸油和动物脂
- **机械润滑油**：气机设备核心耗材，兰芳石油的主要用途之一
- **沥青防腐**：铁甲舰水线以下防腐材料，来源兰芳
- **早期油机（内燃机）**：研发阶段，尚未量产；兰芳石油是这条技术路线的命脉

### 不存在或极罕见的技术（1900年代）
- 飞机：无
- 无线电：极初级实验阶段，不可实战使用
- 汽车：极少数原型机，未普及
- 内燃机武器：尚在试验阶段

### 气机视觉特征（写作参考）
- 外观：铸铁主体 + 黄铜管路 + 榫卯结构补强件 + 宣纸操作图贴在机身侧面
- 声音：低沉的嘶嘶声、有节律的金属击打声、蒸汽泄压的"哧"声
- 气味：机油、煤烟、金属高温气味混合
- 操作：工程师随身携带铜制校准工具，用图纸而非仪表盘判断运行状态

## 三、政治权力结构

```
皇帝
├── 内阁首辅（郑玄机，崇工朝；宣征朝告老还乡，暗中仍有影响）
├── 六部（礼/吏/户/刑/兵/工）
│   └── 工部下设：矿冶司、气机司、船政司、军器司
├── 格物院（皇家技术顾问机构，有品无权，谢长庚任职处）
├── 锦衣卫（情报机构，使用电报密码）
└── 司礼监（内廷太监机构）
    └── 秉笔太监刘恩（掌控紫禁城电报总机房、皇家工厂内务）
```

**权力关系要点**：
- 格物院在朝廷中地位尴尬：皇帝重视，传统官僚轻视
- 郑玄机与刘恩是表面盟友、实际竞争的关系
- 谢长庚是技术官僚，不涉及正常党争，但持有格物总录使他成为各方潜在目标

## 四、人物圣经

### 谢长庚（主角）
- **年龄**：1878年生，故事开始（1900年）22岁
- **外貌**：中等身材，手上有机油难以洗净的纹路，习惯眯眼看机械结构
- **出身**：兰芳省古晋，汉族移民后代，祖父参与抗英，父亲是南洋矿业都司的基层工程师
- **知识结构**：兰芳格物书院毕业，专攻气机推进；读过大量《天工开物》原典；不擅长文言奏折，但能写清晰的技术议文
- **说话方式**：直白、精准、偶尔用类比打比方；不用文言，口语化但不粗俗；遇到技术问题会突然安静下来，盯着某处发呆
- **性格核心**：理工思维主导，先找原因再找方案；对人的恶意反应迟钝，对技术的直觉极敏锐；忠诚但不盲从
- **持有物**：格物总录（第17章后）；祖父战地日记；兰芳来信若干

### 朱载机（第一皇帝，格宗）
- **年龄**：约1860年生，故事开始时40岁；1908年驾崩，约48岁
- **外貌**：手上有茧，指甲缝常有金属屑；穿着随意，在工厂时常摘掉冠帽
- **说话方式**：对话极简，多用疑问句测试对方；面对懂行的人才会展开说；不喜欢被拍马屁
- **核心动作**：翻图纸、调校刻度、在金属片上刻字
- **禁忌**：他从不在朝堂上谈技术，只在工厂里谈；对政治本能厌倦

### 朱载昱（第二皇帝，烈宗）
- **年龄**：约1870年生，1908年即位时38岁
- **外貌**：比兄长高半头，军人体型，目光有压迫感
- **说话方式**：命令式，简短；但私下与信任的人交谈时会放开，有幽默感
- **核心动机**：渴望完成兄长未竟之事；对英国占领兰芳有刻骨仇恨

### 郑玄机（首辅，反派）
- **年龄**：约1845年生，故事开始时55岁
- **外貌**：保养极好，看起来比实际年龄年轻十岁；手指修长，习惯把玩一枚玉扳指
- **说话方式**：文言夹白话，句子里常有三层意思；从不直接否定，只说"此议甚好，然……"；擅长用停顿制造压力
- **行事逻辑**：不亲自动手，只布局；永远在下一步落子前就想好退路
- **弱点**：过于相信自己对人心的判断；低估了技术系统的独立性

### 沈青鸾（郑和遗脉，女主）
- **年龄**：约1882年生，正式出场时约35岁（第二幕）
- **外貌**：中非混血，肤色偏深，汉语带有轻微南洋口音
- **说话方式**：经济简练，不废话；关键信息习惯用问句传递；情绪很少外露
- **知识结构**：通汉语、斯瓦希里语、奥斯曼土耳其语；熟悉三大洲港口水文

### 刘恩（秉笔太监）
- **年龄**：约1860年生
- **说话方式**：宫廷腔，敬语极多，但每句话里都有试探；"奴婢"自称（注意：这是宦官对皇帝的自称，用于内廷场合，不是满清"奴才"文化）
- **行事逻辑**：不站队，只押注；郑玄机是合作者，不是主子

### 陆知微（格物院女博士）
- **年龄**：约1873年生，故事开始时27岁
- **说话方式**：学术腔，引经据典；情绪激动时句子变短、语速加快
- **核心观点**：技术知识应当归公，皇权垄断是技术进步的最大障碍

## 五、禁止出现的内容（世界观一致性红线）

### ❌ 满清相关（绝对禁止）
此世界线中满清从未统治中国，以下内容一律不得出现：

| 类别 | 禁止内容 |
|------|---------|
| 年号/帝号 | 康熙、雍正、乾隆、嘉庆、道光、咸丰、同治、光绪、宣统及任何清代年号 |
| 辫子文化 | 发辫、辫子、剃发、金钱鼠尾；汉族男性无辫（明代发型：束发戴冠） |
| 服饰 | 旗袍（此时间线不存在）、马褂作为正式朝服、八旗服饰 |
| 宫廷礼仪 | "奴才"（满汉通用称呼在此时间线不存在）；宦官自称"奴婢"而非"奴才" |
| 制度 | 八旗制度、旗人/汉人身份区分、文字狱 |
| 人名 | 爱新觉罗家族任何成员；任何清代历史人物姓名 |
| 地名别称 | 盛京（沈阳的清代称呼）等清代专用地名 |

### ❌ 其他时代错误
| 类别 | 禁止内容 |
|------|---------|
| 技术 | 飞机作战、无线电通讯（实战）、量产汽车 |
| 称谓 | "总统"、"共和国"（大明是帝制）；"洋大人"（大明未受过殖民屈辱，不会有这种称呼） |
| 心态 | 民族自卑感、"天朝上国"闭关守旧心态（大明已工业化，对外是自信的竞争姿态）|
| 文化 | 三寸金莲（明代虽有缠足，但工业化后已大幅减少，格物书院女性不缠足）|

### ✅ 正确的明代特征
- 男性：束发，戴网巾或冠帽，工人阶级戴幞头
- 官员服饰：明代补服（有品级兽纹）
- 皇帝自称：朕；臣子自称：臣；太监自称：奴婢
- 朝廷用语：奏折、题本、廷议、圣旨
- 货币：明代银两+铜钱，工业时代已有纸币（大明宝钞升级版）
