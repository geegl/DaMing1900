# 明朝历史常识检查清单

> 本文档用于审校时对照检查，防止出现历史常识错误。所有写作和审校人员必须对照此清单逐项检查。

---

## 一、官制与行政

### 1.1 明朝 vs 清朝官制对照

| 清朝（禁止使用） | 明朝（正确使用） |
|------------------|------------------|
| 军机处 | 内阁 |
| 军机大臣 | 阁臣、内阁大学士 |
| 军机章京 | 内阁中书、翰林编修 |
| 军机扩议 | 廷议、阁议 |
| 军机值房 | 内阁值房 |
| 总理衙门 | 礼部、鸿胪寺 |

### 1.2 内阁权限

- 内阁是**皇帝的秘书咨询机构**，没有法定直接行政权
- 内阁只负责**票拟**（起草意见），不直接下发公文
- 人事任命公文由**吏部**签发，盖吏部印
- 皇帝特批用**宝玺**，由司礼监批红
- **内阁从不在人事公文上盖自己的印章**

### 1.3 官品升迁路径

| 品级 | 标准路径（工部为例） |
|------|----------------------|
| 正六品 | 主事 |
| 从五品 | 员外郎 |
| 正五品 | 郎中 |
| 正四品 | 侍郎 |
| 正二品 | 尚书 |

**注意**：从六品升正五品，标准路径需经过员外郎，破格需皇帝亲批。

### 1.4 任命文书规格

| 品级 | 文书类型 | 形式 |
|------|----------|------|
| 一至五品 | 诰命 | 卷轴式样，材质考究，设香案跪受 |
| 六至九品 | 敕命 | 普通公文形式 |

## 二、称呼与自称

### 2.1 官员互称

| 错误用法（清代/现代） | 正确用法（明代） |
|----------------------|------------------|
| 大人（泛用） | 表字、号、官衔+先生/老先生 |
| 张大人 | 张老先生、张年兄、张侍郎 |
| 谢大人 | 谢老先生、谢年兄、谢主事 |

**注意**："大人"仅用于：
- 下级对极高级别官员（阁老、督抚）
- 平民对官员的敬称
- 同僚间极少互称"大人"

### 2.2 自称

| 场景 | 上级对下级 | 下级对上级 | 平级之间 |
|------|-----------|-----------|----------|
| 正式 | 本官 | 下官、卑职 | 鄙人、愚 |
| 半正式 | 老夫 | 学生 | 直呼己名 |
| 禁止 | 我、本人 | 我、本人 | 我、本人 |

### 3.1 官服颜色（文官）

| 品级 | 颜色 | 补子图案（文官） |
|------|------|------------------|
| 一品 | 绯色（红） | 仙鹤 |
| 二品 | 绯色（红） | 锦鸡 |
| 三品 | 绯色（红） | 孔雀 |
| 四品 | 绯色（红） | 云雁 |
| 五品 | 青色 | 白鹇 |
| 六品 | 青色 | 鹭鸶 |
| 七品 | 青色 | 溪敕 |
| 八品 | 绿色 | 黄鹂 |
| 九品 | 绿色 | 鹌鹑 |

**禁止**：
- 紫色官服（明朝无紫色官服体系）
- 深青色用于三品（三品穿绯色）

### 3.2 日常服饰

- 官员日常穿**直身**、**道袍**
- 赐服：**蟒袍**、**飞鱼服**、**斗牛服**（非品官常服）

## 四、宫廷空间

### 4.1 明朝皇帝主要活动场所

| 场所 | 功能 |
|------|------|
| 乾清宫 | 皇帝寝宫、日常理政核心 |
| 交泰殿 | 存放宝玺 |
| 坤宁宫 | 皇后寝宫 |
| 文华殿 | 经筵讲学 |
| 武英殿 | 斋居、召见大臣 |
| 西暖阁 | 乾清宫西侧暖阁，皇帝休息处 |

**注意**：
- 养心殿在明朝主要用于修仙炼丹（嘉靖）或存放物品
- 皇帝病榻设在**乾清宫**，不是养心殿
- "养心殿西暖阁"是清朝雍正后才形成的政治空间格局

### 4.2 窗户材料

| 建筑等级 | 窗户材料 |
|----------|----------|
| 皇家建筑 | 明瓦（蚌壳/羊角制成）、高丽纸 |
| 普通建筑 | 窗户纸 |

**禁止**：皇家建筑使用普通纸张糊窗

## 五、礼仪制度

### 5.1 皇帝出殡

| 禁止（错误） | 正确做法 |
|--------------|----------|
| 百姓夹道跪拜 | 街道戒严，百姓闭户 |
| 百姓沿路哭泣 | 甲士林立，五步一岗 |
| 穿午门向南 | 出玄武门向北，经德胜门往皇陵 |

### 5.2 茶礼仪

| 禁止（清代习惯） | 正确做法（明代） |

| 茶碗盖刮碗沿发声 | 放下茶盏，轻啜 |

## 七、语言词汇

### 7.1 禁止使用的现代词汇

| 现代词汇 | 明代替代 |
|----------|----------|
| 设计（名词） | 图样、画样、规制、底样 |
| 花边消息 | 坊间秘闻、市井风月、稗官野史 |
| 头版 | 首页、首篇 |
| 全力支持 | 鼎力相助、尽心支应、绝无推诿 |
| 后续的工作 | 后续差使、营造之事 |
| 技术层面 | 技艺之道、格物之术 |
| 系统性 | 统筹、全局 |
| 结构性 | 根本、要害 |
| 日程、日程表 | 起居、行止、起居注 |
| 烂尾 | 中辍、废置、半途而废 |
| 经费不足 | 度支匮乏、帑银不继、户部驳了款项 |
| 流民和破落户 | 逃户与市井畸零之人、流冗 |

### 7.2 商业术语（禁止使用现代商业词汇）

| 现代词汇 | 明代替代 |
|----------|----------|
| 物流成本 | 水陆脚价、转运之费 |
| 转手贸易 | 居间转贩、牙行过载 |
| 利润率 | 获利、盈余 |
| 库存积压 | 压货、货滞仓中 |
| 督抚设卡 | 各处钞关与税监、厘金局 |

### 7.3 太监用语 vs 文官用语

| 场景 | 太监用语 | 文官用语 |
|------|----------|----------|
| 称呼皇帝 | 万岁爷、皇爷 | 皇上、圣上、主上、圣驾 |
| 自称 | 奴婢、奴才 | 臣、下官、卑职 |

**注意**：首辅、尚书等文官**不应使用"万岁爷"**，这是太监/包衣用语。

### 7.4 制造机构命名

| 禁止（现代/游戏感） | 正确（明代官办） |
|---------------------|------------------|
| 工坊 | 制造局、机器局、兵仗局、营缮所 |
| 工坊的事情 | 制造局之事 |

### 7.5 禁止使用的英语单词

| 英语 | 替代 |
|------|------|
| dismiss | 抛诸脑后、置之不理 |
| sabotage | 有意破坏、暗中作梗 |
| 其他英语词汇 | 必须用中文表达 |

### 7.3 医学术语

| 现代词汇 | 古代替代 |
|----------|----------|
| 矽肺 | 尘肺、肺石病 |
| 肺炎 | 肺热、肺火 |
| 癌症 | 症瘕、积聚 |

## 八、元小说错误

### 8.1 禁止

- "第X章那天夜里"——人物不能知道自己在第几章
- "如前文所述"——打破第四面墙
- 任何提醒读者"这是小说"的表达

## 九、审校检查流程

### 9.1 每章必须检查的项目

- [ ] 官制词汇是否正确（无军机处等清朝词汇）
- [ ] 称呼是否正确（无泛用"大人"）
- [ ] 自称是否正确（无滥用"我"）
- [ ] 太监用语与文官用语是否区分
- [ ] 服饰是否合规（无紫色官服）
- [ ] 年号是否正确（无清朝年号）
- [ ] 无英语单词
- [ ] 无现代术语（设计、花边消息等）
- [ ] 无现代商业术语（物流成本、利润率等）
- [ ] 无元小说表达
- [ ] **设定一致性**：与BIBLE.md设定是否冲突（如太子问题）

### 9.2 设定一致性检查要点

**根据BIBLE.md核实：**
- 朱载机（崇工皇帝）**无子嗣**
- 皇位传给弟弟朱载昱（信王）
- 郑玄机在1918年才发动政变，扶植傀儡"朱翼钊"
- 文中不应出现"太子七岁"等与设定冲突的内容

### 9.2 Codex CLI 二审不可跳过

**流程要求**：
1. 写作完成后必须调用 Codex CLI 进行二审
2. 二审发现的问题必须返回修改
3. 不允许自己写审校报告
4. 不允许跳过任何审校步骤
