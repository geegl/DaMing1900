# 第二幕结构总纲

> 范围：`101-280章`
>
> 时间：`1908-1918`
>
> 总主题：**征服世界的史诗与代价**

## 一、第二幕总目标

第二幕要完成 5 件事：

1. 把谢长庚从“第一幕的工程师核心”推进成“远征时代的战争机器中枢”
2. 把兰芳、石油、海军改革、远征全球化真正并轨
3. 把郑和遗脉、东非节点、奥斯曼节点、德国交易等全球网络立住
4. 把帝国外部胜势推到巅峰
5. 在最强胜势处切出国内政变消息，为第三幕完成断崖式转向

## 二、五阶段拆分

### 第一阶段：海军改革与出征前夜（101-130）

- 功能：第二幕起势
- 关键任务：
  - 新君朱载昱完成对第一幕残局的接管
  - 谢长庚转入二皇核心体系
  - 海军改革从争论进入成军与训练
  - 远征目标与国家理由被写实
- 关键感受：
  - 新朝新气象
  - 但所有改革都带着先帝遗产的阴影

### 第二阶段：南洋突破与东非接线（131-170）

- 功能：打开第二幕的地理与文明尺度
- 关键任务：
  - 突破马六甲
  - 兰芳解放
  - 郑和遗脉正式进入主线
  - 达累斯萨拉姆与海外接线建立
- 关键感受：
  - 这是帝国出海，不是单纯舰队远航
  - 历史遗产被重新唤醒

### 第三阶段：苏伊士攻坚与第二公理悟道（171-210）

- 功能：技术破局 + 世界级战争硬仗
- 关键任务：
  - 苏伊士运河争夺
  - 第二公理落地
  - 国内 / 海外双压同步出现
- 关键感受：
  - 工程与战争不再能分开写
  - 主角能力上限被再定义

### 第四阶段：地中海决战与帝国巅峰（211-250）

- 功能：高潮前的胜势与势能拉满
- 关键任务：
  - 地中海舰队决战
  - 帝国外部力量达到最强
  - 多方国际力量全面交错
- 关键感受：
  - 世界真的被大明舰队撬动了
  - 胜利越大，代价越近

### 第五阶段：炮击伦敦与国内政变消息（251-280）

- 功能：第二幕高潮 + 第三幕引爆器
- 关键任务：
  - 炮击伦敦
  - 英伦失守
  - 国内政变消息传来
- 关键感受：
  - 外部达到巅峰
  - 内部同时塌陷
