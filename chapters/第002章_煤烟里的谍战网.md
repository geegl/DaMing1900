# 第002章 煤烟里的谍战网

第二天早上，蒸汽管道爆了。

陈铁正在吃早饭——一碗掺了煤灰的糙米粥——突然听到一声巨响。

"轰！"

整个工厂都震了一下。蒸汽从车间东北角喷涌而出，工人们四散奔逃。

"管道炸了！管道炸了！"

"快跑！高压蒸汽会把人烫熟的！"

陈铁站起来，看向东北角。

脑海中，幽蓝色的数据流自动浮现——

**【天工差分机·危机解析】**
```
威胁等级：高
破裂位置：第3节管道（与昨夜解析一致）
蒸汽压力：180psi
预计扩散范围：半径15米
伤亡风险：78%
```

陈铁握紧生铁扳手。

"第3节管道..."他低声说，"昨晚就看到了。"

---

施罗德带着两个德国工程师冲进车间。

"快！关闭主阀门！"施罗德用德语吼道。

工程师们手忙脚乱地操作，但蒸汽还在喷涌。

"主阀门卡死了！"一个工程师喊道，"锈死了！"

施罗德脸色铁青："用扳手！砸开！"

工程师们拼命砸，但阀门纹丝不动。

蒸汽越来越浓，能见度降到不足两米。

---

工人们躲在远处，没人敢靠近。

王鞭子冲过来，看到陈铁还在原地，一鞭子抽在他背上："你愣着干什么！滚开！"

陈铁没动。

他看着那团喷涌的蒸汽，脑海中数据流疯狂滚动——

**【天工差分机·紧急修复方案】**
```
方案一：更换阀门（耗时：3小时）✗
方案二：侧向泄压（耗时：30分钟）✗
方案三：精准爆破（耗时：3分钟）✓
```

陈铁眼睛一亮。

"精准爆破？"

**【详细步骤】**
```
1. 定位：第4节管道上方的泄压阀
2. 力度：用扳手敲击3次，角度45度
3. 效果：泄压阀自动打开，压力降低60%
```

陈铁攥紧扳手。

"只有3分钟..."

---

他冲进蒸汽里。

"陈铁！你疯了！"王鞭子吼道。

但陈铁已经消失了。

蒸汽呛得他睁不开眼，温度至少有120度。皮肤被烫得发红，但他咬牙往前冲。

脑海中，幽蓝色的数据流像灯塔一样指引方向——

**【导航】**
```
前进：5米
左转：30度
前进：3米
目标：泄压阀（距离2米）
```

陈铁按照指示前进，终于摸到了泄压阀。

阀门已经被锈死了，但他知道该怎么开。

他举起扳手，对准阀门上方45度角——

"当！当！当！"

三下精准的敲击。

---

"嗤——"

一声尖锐的嘶鸣。

泄压阀打开了。

高压蒸汽从侧面喷出，管道内的压力迅速下降。

主阀门的压力减轻，不再卡死。

德国工程师趁机转动主阀门，彻底关闭了蒸汽。

车间里，终于安静下来。

---

陈铁从蒸汽里走出来，浑身湿透，衣服被烫出了几个洞，皮肤红肿。

但他还活着。

施罗德愣住了。

他看着陈铁，用德语对旁边的工程师说："这不可能...那个泄压阀已经锈死了十年。他是怎么打开的？"

工程师摇头："我不知道...就算是老师傅，也不可能那么精准地找到敲击点。"

施罗德盯着陈铁，眼神复杂。

"你..."他用生硬的中文说，"怎么知道敲那个位置？"

陈铁低头，没说话。

"运气好。"他说。

施罗德没说话，转身走了。但他走了两步，又回头看了陈铁一眼。

那一眼，带着审视。

---

陈铁回到工棚，刚坐下，就听见身后传来一个沙哑的声音。

"运气好？"

陈铁转头。

一个枯瘦的老人站在他身后，左臂缺失，袖管空荡荡的。

老鬼。

锅炉房的老工头，在这里干了三十年。

陈铁见过他几次，但从没说过话。

"你修管道的方式，不对劲。"老鬼说。

陈铁紧张："我只是...敲了几下。"

老鬼笑了，露出黄黑的牙齿："泄压阀锈死了十年。你敲了三下，就开了。你的眼睛...看到了什么，对吧？"

陈铁沉默。

老鬼从怀里掏出一张泛黄的纸，递给陈铁。

"你能看懂这个吗？"

---

陈铁接过纸，看了一眼。

脑海中，数据流再次浮现——

**【天工差分机·图纸解析】**
```
类型：差分机核心结构
功能：多项式计算
复杂度：极高
缺陷：第12号齿轮磨损
制造者：德国西门子公司
```

陈铁指着图纸："这里，第12号齿轮有问题。"

老鬼的瞳孔收缩。

"你...真的能看到。"

老鬼坐下来，压低声音："你知道这是什么吗？"

陈铁摇头。

"这是差分机。"老鬼说，"洋人用来算东西的机器。算炮弹的落点，算商业的账目，算...很多东西。"

"你手里那张图，是差分机的核心结构。但我搞不懂它的原理。"

老鬼看着陈铁："但你能看懂它的缺陷。"

陈铁握紧那张纸，手心出汗。

他不能说出那些幽蓝色的字迹，那是他保命的底牌。

"我...我能'看'透它们。"他斟酌着词句，低声说，"当我看机器的时候，我能感觉到它们哪里病了，哪里快断了。我就是...知道。"

老鬼看着他，倒吸了一口凉气。

这是传说中的"天工之心"，是对机械有着神明般直觉的天才。

老鬼沉默了很久。然后他笑了。

"好。"老鬼说，"从今天起，你跟我。"

---

老鬼带陈铁去了一个地方。

天津卫城北，一座破败的土地庙。

老鬼推开庙门，里面不是神像，而是一间密室。

墙上挂着地图，桌上堆着文件，角落里放着几台奇怪的仪器。

"你知道义兴会吗？"老鬼问。

陈铁摇头。

"我们是一个地下情报网，专门对付洋人。"老鬼说，"我们在工厂、码头、海关...到处都有人。"

"我们在监视他们，记录他们，破坏他们。"

老鬼拿出一张纸："这是你要做的第一件事。"

陈铁接过纸。

上面写着：**施罗德的办公室，明天晚上，拿走这份文件。**

旁边还画着一张简图——施罗德办公室的位置。

"为什么？"陈铁问。

"因为施罗德不只是监工。"老鬼说，"他在掩盖一笔烂账。"

"阿牛的死，还有今天管道的爆炸，都不是意外。是有人故意用了劣质的生铁，在吸大明军工的血！施罗德的办公室里，有他们走私回炉铁的账本。"

陈铁猛地抬起头。

"如果你拿到账本，"老鬼看着他的眼睛，"阿牛的仇，就能报。"

陈铁握紧那张纸。

"好。"他说。

---

老鬼看着陈铁，眼神复杂。

"你有特殊的能力。"老鬼说，"但你要记住，这个能力，不是用来炫耀的。"

"是用来保护自己，保护兄弟，保护大明。"

"如果你暴露了，洋人会杀了你。朝廷会杀了你。连我们的人，都会杀了你。"

陈铁点头。

"我明白。"

老鬼拍拍他的肩膀："明天晚上，我会在外面接应你。"

"记住，只拿文件，不要杀人。我们还不到那个时候。"

陈铁看着手中的纸，低声说："好。"

---

**第002章 完**
