# 第003章 劣质炮的真相

1899年十月十二，夜。

陈铁站在施罗德的办公室门外。

工厂已经安静下来，只剩下远处蒸汽机的低鸣。

老鬼在院子里等他。

"记住，"老鬼低声说，"只拿账本，不要动其他东西。"

陈铁点头。

他看向办公室的门锁。脑海中，幽蓝色的数据流浮现——

**【天工差分机·锁具解析】**
```
类型：德国制锁（西门子产）
等级：高
弱点：第3号锁舌有0.1mm偏差
开锁方式：用铁丝插入，逆时针旋转45度
耗时：约5秒
```

这种德国最新的防盗锁，据说连天津卫最厉害的神偷都要研究三天。

但在陈铁眼里，它就像一个被剥光了衣服的婴儿。

陈铁从怀里掏出一根铁丝——老鬼给他的。

他插入锁孔，逆时针旋转。

"咔嗒。"

门开了，仅仅用了三秒。

---

办公室里很暗。

陈铁没有点灯。他借着窗外的月光，看到桌上堆满了文件。

他走到桌前，开始翻找。

脑海中，数据流自动扫描着每一份文件——

**【天工差分机·文件扫描】**
```
文件1：蒸汽机采购清单（无关）
文件2：工人工籍表（无关）
文件3：炮管原材料采购单（重点）
```

陈铁拿起第三份文件。

月光下，他看到文件上写着：

**"炮管原材料采购单"**

**"供应方：钱四海商行"**

**"材质：回炉铁（低价）"**

**"数量：300根"**

**"验收人：施罗德"**

陈铁的手开始发抖。

脑海中，数据流疯狂涌现——

**【天工差分机·原材料解析】**
```
材质：劣质回炉铁
含硫量：超标200%
砂眼率：超标300%
抗压强度：低于标准40%
炸膛概率：78%
致死率：99%
```

陈铁猛地抬起头。

**99%的致死率。**

这意味着，每一根用这种铁造的炮管，开火时都有78%的概率炸膛。

而炸膛，99%会杀死炮手。

阿牛的死，不是意外。

那些被劣质炮管炸死的士兵，不是意外。

**这是谋杀。**

---

陈铁继续翻找。

他在抽屉里找到了另一份文件：

**"回炉铁采购账本"**

账本上记录着每一笔交易：

- 1896年（1896年）：采购回炉铁500根，单价3两白银/根，售予兵部，售价8两白银/根
- 1897年（1897年）：采购回炉铁800根，单价2.5两白银/根，售予兵部，售价7.5两白银/根
- 1898年（1898年）：采购回炉铁600根，单价2.2两白银/根，售予兵部，售价7两白银/根
- 1899年（1899年）：采购回炉铁300根，单价2两白银/根，售予兵部，售价7两白银/根

**利润：每一根炮管，净赚4-5两白银。**

**四年总利润：约10000两白银。**

陈铁看着那个数字。

8000两白银。

相当于240万文铁钱。

相当于阿牛这样的工人，不吃不喝工作2500年。

陈铁攥紧账本，手指发白。

---

他继续翻。

在账本的最后一页，他看到了一个名字：

**"分红名单"**

- 钱四海：60%（6000两）
- 张廷远：30%（3000两）
- 施罗德：10%（1000两）

陈铁停下。

**张廷远。**

这个名字他听过。

兵部尚书，朝廷大员，天工帝最信任的臣子之一。

原来...

**原来这才是真相。**

这不是简单的贪污。

这是一个从兵部尚书，到商人，到洋人监工，一层层盘剥的产业链。

劣质炮管 → 兵部采购（张廷远批准）→ 钱四海供货 → 施罗德验收 → 工厂生产

最后，炮管炸膛，死的是士兵，是像阿牛这样的工人。

而张廷远、钱四海、施罗德，他们分着10000两白银的利润。

---

陈铁把账本放回怀里。

他刚要离开，突然看到桌上还有一份文件。

文件的封面上写着：

**"兰芳橡胶林采购计划"**

陈铁愣了一下。

兰芳。

阿牛生前说过的地方。

"兰芳的海是蓝色的，没有煤烟，没有铁锈味。"

陈铁拿起文件，翻开。

**【天工差分机·文件扫描】**
```
类型：兰芳橡胶林采购计划
供应方：钱四海商行
买家：兵部、工部
金额：50万两白银
备注：已与英国东印度公司、荷兰殖民政府达成协议，共同开发兰芳橡胶资源
```

陈铁看到最后一句——

**"共同开发兰芳橡胶资源"**

他突然想起了什么。

表哥陈铜在兰芳。

表哥说，兰芳的华人正在被洋人屠杀。

原来...

**原来这是钱四海在卖兰芳。**

---

陈铁把文件放回去。

他悄悄退出办公室，关上门。

院子里，老鬼在等他。

"拿到了吗？"老鬼低声问。

陈铁点头，从怀里掏出账本。

老鬼接过账本，借着月光看了一眼。

他的脸色变了。

"张廷远..."老鬼低声说，"兵部尚书。"

"这牵扯到朝廷大员。"

陈铁沉默。

老鬼合上账本，看向陈铁："你现在知道了。"

"这不是简单的贪污。"

"这是系统性的卖国。"

"劣质炮管，让大明的军队失去战斗力。"

"兰芳橡胶，让洋人控制了大明的资源。"

"而我们，"老鬼指了指账本，"只是他们赚钱的工具。死了，也只是一个编号。"

陈铁握紧拳头。

"我要杀了施罗德。"他说。

"不。"老鬼说，"施罗德只是棋子。真正的敌人，是钱四海，是张廷远，是整个卖国产业链。"

"你要报仇，就要先活下来。"

"然后，把他们的骨头一寸一寸抽走。"

陈铁看着老鬼。

"怎么做？"他问。

老鬼笑了。

"从今天起，你就是义兴会天津卫情报网的成员。"

"你的任务，是继续调查钱四海。"

"找到他更多的罪证。"

"等到时机成熟，"老鬼压低声音，"宫里很快就会派人来查账。那是东厂的太监。他们和张廷远不是一路人。"

"我们要让这份账本，出现在东厂的案头上。借他们的刀，杀张廷远的人。"

陈铁点头。

"好。"他说。

---

**第003章 完**
