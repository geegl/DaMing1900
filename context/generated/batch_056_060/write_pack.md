# 第56-60章批次写作包

## 阶段位置

- 第一幕第六段后半，1905 年前后
- 谢长庚已因新舰提速进入公开视野
- 军工晋升正在转化为身份模糊与政治风险
- 朱载机病势开始真正压到技术线之上
- 郑玄机尚未亲自下场，但已通过文书、密档、清查等方式提前收口

## 本批核心任务

- 写出“看得见的功”和“看不见的名分”之间的张力
- 把格物院与密档体系拉进主冲突
- 让陆知微、刘恩、朱载机三方从不同角度把思想线和权力线顶起来
- 为第60章清查全面铺开制造持续压迫感

## 逐章目标

- 第56章《看不见的加官》：写谢长庚名声大起却名分暧昧
- 第57章《密档房的锁》：写密档提前收口与《总录杂记》首次现影
- 第58章《陆知微的火气》：写技术归公思想第一次公开撞墙
- 第59章《皇帝的沉默》：写朱载机病重与提前托付的气味
- 第60章《密档清查》：把政治高压正式压到谢长庚头上

## 写法要求

- 延续工业细节与制度冷感并行的写法
- 不把政治写成空喊阴谋，要落在文书、钥匙、锁、名分、签押这些具体物上
- 谢长庚仍然先从做事视角理解问题，再慢慢意识到自己已被卷进更大的结构
