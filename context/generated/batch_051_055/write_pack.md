# 第51-55章批次写作包

## 阶段位置

- 第一幕中后段，时间进入 1904-1905 年
- 谢长庚已从单纯查兰芳油线，转入军工体系内部的制度性阻力
- 朱载机开始把他当成能托更重事情的人
- 郑玄机尚未正面对谢长庚出手，但利益链已经先动起来

## 本批核心任务

- 把“技术改进会动到钱路”这层冲突写实
- 把谢长庚从幕后匠人推到公开军工核心位置
- 让承包商、验印、供应链、官样文章形成一条完整阻力链
- 用一次海试成功，为后续更高层的政治视线埋钩

## 逐章目标

- 第51章《锅炉房的新法》：从锅炉与润滑联动切入，说明真正的提速来自脏活细活
- 第52章《承包商们》：正面引出油料与军工承包商的利益圈
- 第53章《慢半拍》：用工程排查锁定人为做手脚的润滑油
- 第54章《谁签的验印》：把问题追进制度与验印链条
- 第55章《新舰提速》：公开海试成功，谢长庚被朝廷正式看见

## 写法要求

- 继续保持《高堡奇人》式异史日常感，不解释世界观
- 工业细节要具体，优先写声音、温度、油膜、金属感
- 冲突要落在人、账、流程、签押这些细处，不要空喊大阴谋
- 谢长庚仍然是工程师思路：看痕迹、测数据、拼系统
- 每章结尾必须给出下一章的明确推进钩子
