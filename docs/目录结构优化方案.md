# 《大明1900》目录结构优化方案（I-Lang版）

---

## 一、当前目录问题

```
问题：
1. docs/目录层级过多（6层）
2. memory/文档冗余
3. progress/空文件
4. chapters/已清空
5. automation/脚本混杂
```

---

## 二、优化目标

```
目标：
1. 文档扁平化（≤3层）
2. memory单一化（1个文件）
3. 脚本分类化（按功能）
4. 状态文件集中化
```

---

## 三、I-Lang优化流程

```
[READ:current_structure]=>
[ANALYZE:redundancy]=>
[PLAN:new_structure]=>
[EXECUTE:reorganize]=>
[VERIFY:completeness]=>
[COMMIT:git]=>
[OUT:result]
```

## 五、优化动作

```
[DELETE:冗余文档] =>
```
