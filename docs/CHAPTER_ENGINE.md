## 🔒 零、执行铁律（最高优先级）

### 【时间锁法则】
> **本章的时间跨度不得超过 X 小时。**

| 章节类型 | 时间锁 | 效果 |
|---------|--------|------|
| 普通章节 | ≤2小时 | 迫使AI描写细节 |
| 高潮战役 | ≤10分钟 | 慢镜头，每秒必争 |
| 特殊跳跃 | 必须标注 | 用"三年后"等明确语句 |

### 【POV绝对死锁】
> **一章只能有一个"摄像头"，绑在一个人身上。**

| 允许 | 禁止 |
|------|------|
| ✅ 通过听觉推测他人 | ❌ 写其他人物内心想法 |
| ✅ 通过视觉观察行为 | ❌ 切换到其他场景 |
| ✅ 通过对话获取信息 | ❌ 全知全能视角 |

### 【四位大师审查核心修复】

1. **金手指禁令**：陈铁没有任何超能力
2. **POV前30章死锁**：只允许陈铁和老鬼
3. **爽点前置**：前三章必须有微小爽点
4. **日常细节强制**：每章300字（煤烟、铁锈味、震聋的耳朵）
5. **洪威帝存活规则**：第二部结尾（第155章）死去

## ⚙️ 一、章节结构（15-40-30-15法则）

### 1. 【开篇Hook】(15%) - 10种强力开头

| 技巧 | 说明 | 示例 |
|------|------|------|
| **动作切入** | 绝境起手 | "第三根高压蒸汽管爆裂的时候，老鬼的左脸已经被烫掉了一层皮。" |
| **感官轰炸** | 嗅觉/触觉 | "天津卫的冬天，风里只有煤渣子和劣质机油的焦臭味。" |
| **不祥倒计时** | 时间压力 | "距离大英皇家舰队主炮充能完毕还有120秒。" |
| **诡异细节** | 暗示阴谋 | "太和殿前的铜缸里，漂浮着一层原油。" |
| **嚣张宣言** | 立住人设 | "在我的领海里，洋人也别想摸一下大明的死狗。" |
| **屈辱通牒** | 憋屈起手 | "瓦尔德的电报只有八个字：交出军机图，或受洗。" |
| **微观隐喻** | 小动作隐喻 | "张廷远将英镑扔进炼钢炉。" |
| **残酷献祭** | 死人开局 | "定远号殉爆的光芒还没散去。" |
| **阶级错位** | 材质反差 | "波斯地毯上扔着沾满黑泥的铸造靴。" |
| **截获情报** | 谍战悬疑 | "大明暗码：内鬼在南书房。" |

### 2. 【施压推演】(40%) - 冲突展开

- 对话交锋、情报汇集、敌人步步紧逼
- 主角处于"憋屈"状态，积蓄情绪压力

### 3. 【高潮破局】(30%) - 物理爆发

- 拔刀、开炮、按下按钮、掀翻桌子
- 必须用极度写实和粗糙的重工业物理动作

### 4. 【收网悬念】(15%) - 10种结尾钩子

| 钩子 | 说明 |
|------|------|
| **物理级过载警报** | 气压表崩断在红区 |
| **致命电文残片** | 染血纸带吐出惊悚名字 |
| **巨兽破雾** | 敌方恐怖战争兵器出现 |
| **工业烙印** | 发现"自己人"的制造印记 |
| **生铁两难** | 两个目标在同一引爆器上 |
| **骤然死寂** | 轰鸣声突然消失 |
| **撕裂伪装** | 唯诺者熟练拉开毛瑟枪保险 |
| **暴君反问** | 狂妄金句收尾 |
| **历史倒计时** | "距离除夕惊变还有48小时" |
| **视听切黑** | 在动作发生前零点一秒结束 |

## 📐 二、大纲模板（四冲程）

### 基础参数
- **章节编号**：[第X章]
- **章节标题**：[动词+重工名词]
- **POV**：[本章视点人物]
- **时间地点**：[精确到时辰和工业环境]
- **核心材质锁**：[本章高频视觉材质]

### 四冲程分场

| 冲程 | 篇幅 | 内容 |
|------|------|------|
| **进气阀** | 0-15% | Hook开场，动作/悬念切入 |
| **压缩机** | 15-55% | 极限施压，敌人逼近 |
| **点火舱** | 55-85% | 物理爆发，暴烈破局 |
| **排气管** | 85-100% | 致命悬念，切黑收尾 |

## 💾 三、黑匣子存档

生成正文后，必须输出状态更新：

```
🩸 物理磨损：[谁受伤了？伤在哪？]
📦 战备消耗：[什么道具被消耗？]
🪝 悬念遗留：[本章末尾留下了什么悬念？]
```
