## 文档层级说明

| 层级 | 名称 | 说明 | 更新频率 |
|------|------|------|---------|
| **00-宪法层** | 世界观 | 核心设定，一旦确定不可变 | 极少更新 |
| **01-规划层** | 大纲 | 章节规划，人物数据库 | 每批次更新 |
| **02-执行层** | 引擎 | 写作引擎，流程控制 | 稳定不变 |
| **03-参考层** | 参考 | 写作参考，政治/历史/文本 | 按需补充 |
| **04-质控层** | 质控 | 验证规则，质量报告 | 每章验证 |
| **05-写作指南** | 指南 | 教训总结，技巧备忘 | 持续迭代 |
| **06-参考资料** | 原文 | 原始参考文本 | 不更新 |
