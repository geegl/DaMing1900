# 番茄小说签约要求（初步提取）

**来源**: AI写小说-番茄1月新规和过稿Prompt

---

## 一、番茄小说核心规则（2025年1月新规）

### 1. 字数节点

| 节点 | 字数要求 | 说明 |
|------|---------|------|
| **申请签约** | 2万字 | 达到后可申请签约 |
| **推书荒** | 3万字 | 慎重使用 |
| **验证期** | 8万字 | 进入验证期 |
| **首秀** | 10万字 | 进入首秀期 |
| **全勤奖** | 10万字 + 听读≥500 | 必须同时满足3条 |
| **多书名实验** | 20-50万字 | 获得多书名测试资格 |

### 2. 全勤奖要求（新规）

| 日更字数 | 月奖励 | 要求 |
|---------|--------|------|
| 4000字/日 | 600元/月 | 10万字 + 听读稿费过500 + 日更4000-6000字 |
| 6000字/日 | 800元/月 | 同上 |

**三条必须同时满足**：
1. 作品达到10万字
2. 听读稿费过500元
3. 每天更新4000-6000字

### 3. 追更人数新定义（2025年1月2日）

**旧定义**: 阅读到最近3日发布章节的读者数量

**新定义**: 阅读最近7日更新章节的有效读者数

## 二、首秀关键要求

### 1. 首秀前准备

- ✅ **建议存稿到14万字**
- ❌ **禁止爆更**: 会打乱推荐机制
- ❌ **禁止断更**: 更可能直接腰斩给量

### 2. 首秀期间（3周左右）

| 要求 | 说明 |
|------|------|
| **日更字数** | 每天6000字，稳定定时更新 |
| **千万不能断更** | 一旦断更，流量直接腰斩，很难挽救 |
| **更新时间** | 避开22:00-24:00（卡审，易腰斩）|
| **完读率目标** | 十万字读完破20%，机会更大 |
| **流量判断** | 首秀前几天不来流量，看完读率是否>10% |
| **不要大改作品** | 如连续一周不给流量，可修改简介（300-400字）|
| **心态控制** | 最好不要盯着数据，每天拼命写 |

### 3. 首秀流量问题处理

- **流量不断攀升但很快降下**: 继续坚持写
- **一两周不给流量**: 发书荒试试
- **推流到一定流量池**: 会相对稳定一段时间
- **快书名测时**: 在读人数会明显降低

### 4. 书测（20万字）

**首秀不理想，可以冲20万字书测**：
- 准备5个书名和封面
- 推荐分发
- 测试不同书名的数据效果

## 三、存稿和更新策略

### 1. 开文时间

- ✅ 月初、月底开文，利于冲全勤奖

### 2. 章节字数

- ✅ 每章建议2050-2300字（利于有声小说和冲奖励）

### 3. 日更策略

| 每天码字 | 正常发布 | 存稿 |
|---------|---------|------|
| 4000-8000字 | 2000-6000字 | 余下的存稿 |

### 4. 开书前几天

- 日更4000-6000字
- 争取5天内达到签约标准

### 5. 月底前5天

- 连续日更4000-6000字
- 保证当月最后一天达到10万字以上
- 利于下月冲全勤

## 四、其他重要规则

### 1. AI内容

- **版权**: 属于自己
- **限制**: 只要不抄袭仿写，结果好，平台不会管是否AI
- **建议**: 用AI检测工具测试

### 2. 千万别掉马甲

- 易被恶意举报进小黑屋
- 被很多人搜索点击却不追读
- 会导致留存变差，给量减少

### 3. 新人作者

- 第一本扑的比较多（其实是没了解规则）
- 第二本成绩会有提升
- 不推流但完读率高的要坚持更新
- 破百万作品更吃香（前提是数据不错）

## 五、番茄过稿模板核心

### 1. 大纲模板要求

| 结构 | 要求 |
|------|------|
| **类型和签约难度** | 明确定义小说类型及签约难度 |
| **简纲** | 50字简纲，涵盖情感驱动和情节驱动 |
| **人物设定** | 强调深度、亮点和对比 |
| **创新点** | 流行梗+独创心机点=老套路+新设定 |
| **看点** | 核心梗、热门元素、吸引人的地方 |

### 2. 建议体系

- 明确目标对象和用途
- 分门别类（提高可操作性、增强逻辑性等）
- 每类3-5条具体建议
- 建议间有内在逻辑联系
- 针对性强、可操作性强
- 从不同角度（语法、语义、逻辑）

## 六、关键发现（对《大明1900》的影响）

### ⚠️ 严重问题

| 问题 | 影响 |
|------|------|
| **当前字数不足** | 前10章仅4万字，需达到2万字才能申请签约 |
| **章节字数不符** | 当前4200字/章，建议2050-2300字/章 |
| **日更字数不足** | 当前创作速度无法保证日更4000-6000字 |
| **未考虑首秀节奏** | 没有14万字存稿，首秀期风险高 |

### ✅ 优势

| 优势 | 说明 |
|------|------|
| **质量高** | 文学质量高，可能完读率好 |
| **世界观完整** | 220章大纲完整，可持续更新 |
| **创新性强** | 明朝蒸汽朋克+工业化吃人设定新颖 |
