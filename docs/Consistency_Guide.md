## 一、一致性的7个维度

### 1. 时间线一致性

**问题类型**：
- 时间倒流
- 时间跳跃
- 时间穿越

**防护措施**：
- ✅ timeline_validator_v2.py（时间线审计Agent）
- ✅ 每章强制标注时间
- ✅ 自动验证时间连续性

---

### 2. 世界观一致性

**问题类型**：
- 清朝元素（年号、官职、服饰）
- 现代元素（手机、地铁）
- 技术代差错误

**防护措施**：
- ✅ worldview_validator.py（世界观验证脚本）
- ✅ 禁用词列表（清朝年号、官职、现代元素）
- ✅ 技术设定文档

**检测项**：
```python
FORBIDDEN_QING_TERMS = [
    "光绪", "宣统", "咸丰", "同治", "乾隆", "康熙", "雍正",  # 清朝年号
    "总督", "巡抚", "军机处", "军机大臣",  # 清朝官职
    "辫子", "旗袍", "马褂",  # 清朝服饰
]

FORBIDDEN_MODERN_TERMS = [
    "手机", "电脑", "互联网", "电视", "地铁", "飞机",  # 现代元素
]
```

---

### 3. 人物一致性

**问题类型**：
- 人物特征突变（年龄、外貌、性格）
- 伤痕消失/转移
- 能力突然获得/消失
- 人物关系混乱

**防护措施**：
- ✅ character_physical_profiles.json（人物物理档案）
- ✅ character_voice_profiles.json（人物语言特征库）
- ✅ state.json（人物状态追踪）
- ⚠️ **需要增强：人物特征Agent**

**必须追踪的人物状态**：

| 状态类型 | 示例 | 更新频率 |
|---------|------|---------|
| **物理状态** | 陈铁左腿烫伤（第3章）| 永久伤痕，不可消失 |
| **装备状态** | 老鬼的15公斤大锤 | 获得/失去时更新 |
| **技能状态** | 陈铁的差分机能力 | 获得时记录，不可消失 |
| **关系状态** | 老鬼→陈铁（师徒）| 关系变化时更新 |
| **位置状态** | 陈铁在天津卫 | 移动时更新 |
| **资产状态** | 陈铁存钱220文 | 变化时更新 |

---

### 4. 剧情逻辑一致性

**问题类型**：
- 前后剧情矛盾（施罗德被抓vs死亡）
- 人物行为不合理
- 因果关系断裂

**防护措施**：
- ✅ Codex审查（深层逻辑）
- ✅ fЭershadowing_ledger.json（伏笔账本）
- ⚠️ **需要增强：剧情逻辑Agent**

**检测方法**：
1. 提取章节中的关键事件
2. 对比前文，检查是否矛盾
3. 验证人物动机是否合理
4. 验证因果关系是否成立

---

### 5. 设定一致性

**问题类型**：
- 技术设定矛盾（差分机能力突变）
- 经济设定矛盾（铁钱氧化规则）
- 官制设定矛盾（三司制vs总督制）

**防护措施**：
- ✅ Daming1900_Bible.md（世界观圣经）
- ✅ Reference.md（参考手册）
- ⚠️ **需要增强：设定验证Agent**

**核心设定（不可违反）**：

| 设定类别 | 核心规则 |
|---------|---------|
| **货币设定** | 铁币1年内氧化为铁粉 |
| **技术代差** | 大明落后西方8-20年 |
| **官制设定** | 三司制（布政使、按察使、都指挥使） |
| **阶级设定** | 工籍烙印、贱籍制度 |
| **年号计算** | 天工X年 = 1890 + X |

---

### 6. 文风一致性

**问题类型**：
- AI痕迹词汇（微微、淡淡、心头一颤）
- 文风突变（冷硬→抒情）
- 语言风格不统一

**防护措施**：
- ✅ quality_checker.py（文风检查）
- ✅ character_voice_profiles.json（人物语言特征库）

**禁用词汇表**：
```python
AI_FLAVOR_WORDS = [
    "微微", "淡淡", "心头一颤", "双目赤红", 
    "嘴角勾起一抹弧度", "总而言之", "仿佛"
]
```

---

### 7. POV一致性

**问题类型**：
- POV越界（上帝视角）
- 心理分析出现在第三人称限制视角
- 信息不对称被破坏

**防护措施**：
- ✅ quality_checker.py（POV检查）
- ⚠️ **需要增强：POV守护Agent**

**POV规则**：
- 前期：陈铁单视角
- 中期：可增加老鬼、林霜降等视角
- 每章必须只有一个POV角色
- POV角色只能描述自己看到/听到的，不能知道其他人的心理

   ↓

   ├─ 提取关键事件

3. ❌ **禁止人物特征突变**（伤痕消失、能力消失）

5. ❌ **禁止设定矛盾**（货币、技术、官制）
6. ❌ **禁止AI痕迹词汇**
