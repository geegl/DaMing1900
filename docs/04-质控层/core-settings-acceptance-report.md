## 一、除锈剂体系：三重压迫机制

### 1. 经济层面

| 设定项 | 内容 |
|--------|------|
| **催化氧化机制** | 工部在铁币中掺杂加速氧化催化剂，30天内化为铁粉 |
| **除锈剂成本** | 劣质除锈剂：10文/月（占陈铁收入67%） |
| **收支缺口** | 收入15文/月 - 饭钱5文 - 除锈剂10文 = **-50文/月** |
| **生存状态** | 永远无法攒钱，必须在黑市找额外收入 |

**数学自洽性**: ✅ **彻底解决**  
- 不存在"攒钱125年"的数学悖论
- 经济压迫具象化为**资源消耗战**

#### 义肢依赖症（新设定）

| 设定 | 内容 |
|------|------|

| **隐喻** | **生物学奴役**——底层被技术控制在生存线上 |

## 二、技术壁垒：绝对鸿沟

### 1. 第10章重构核心

#### 陈铁的工具与行动

| 项目 | 内容 |
|------|------|
| **工具** | 粗糙的生铁扳手（把手缠着破布） |
| **能力** | 听出第三齿轮异响位置（天赋） |
| **行动** | 尝试用扳手拆卸齿轮 |
| **结果** | 触发防差错机制，齿轮逆转 |

#### 手指夹伤细节

| 伤势 | 描写 |
|------|------|
| **指甲** | 脱落，露出血肉 |
| **指骨** | 露出，发白 |
| **疼痛** | 钻心剧痛，冷汗直流 |
| **后遗症** | 无法握拳，留下永久性疤痕 |

**动作描写示例**：
> 陈铁咬着牙，用生铁扳手卡住第三齿轮的螺母。
>
> 他用力一拧。
>
> "咔——"
>
> 齿轮突然逆转。
>
> 扳手滑脱，齿轮的齿尖狠狠夹住了他的右手食指。
>
> "啊——"
>
> 陈铁叫出声。
>
> 食指的指甲被夹掉，血涌出来。指骨露在外面，发白。
>
> 他蹲在地上，右手食指血肉模糊，冷汗从额头流下来。
