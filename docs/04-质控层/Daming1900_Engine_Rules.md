## ⏰ 时间锁法则（The Time Lock Rule）

### 核心原则

**"逼着AI磨洋工出细活。"**

AI有一种不可遏制的本能——"急于推进剧情"。时间锁物理锁死时间跨度，迫使AI描写细节。

### 规则

| 章节类型 | 时间锁 | 强制描写 |
|---------|--------|---------|
| **普通章节** | ≤2小时 | 齿轮摩擦、汗水流进眼睛、微表情 |
| **高潮战役** | ≤10分钟 | 慢镜头、每秒必争、每帧描写 |
| **特殊跳跃** | 必须标注 | 用"三天后"等明确标记 |

### 执行示例

**错误示范（无时间锁）**:
> 陈铁走进车间，看到老鬼。他们聊了一会儿，然后陈铁修好了机器。一天过去了。

**正确示范（有时间锁）**:
> 陈铁走进车间。空气中弥漫着煤烟和机油的味道。他的脚步声在铁板上回响。
>
> 老鬼正在一台生锈的蒸汽机前，断掉的左臂袖管在风中晃动。
>
> "过来。"老鬼的声音像砂纸磨过铁板。
>
> 陈铁走过去，蹲下。他的手指触碰到冰冷的蒸汽管道，指尖传来微弱的震动。

## 🎥 POV绝对死锁（POV Absolute Lock）

### 核心原则

**"一章只能有一个摄像头。"**

群像叙事最容易犯"POV跳跃综合征"——上一段写陈铁砸机甲，下一段跳去写张廷远害怕。这会撕碎读者的情绪代入感。

### 规则

| 允许 | 禁止 |
|------|------|
| ✅ 通过听觉推测他人 | ❌ 写其他人物内心想法 |
| ✅ 通过视觉观察行为 | ❌ 切换到其他场景 |
| ✅ 通过对话获取信息 | ❌ 全知全能视角 |

### 执行示例

**错误示范（上帝视角）**:
> 陈铁砸向机甲，心中充满了愤怒。与此同时，张廷远在紫禁城里，他恐惧地想着："这群暴民必须被镇压。"

**正确示范（陈铁POV）**:
> 陈铁举起大锤，砸向机甲的膝盖关节。
>
> 远处传来枪声。他不知道发生了什么，只能猜测张廷远的卫队到了。
>
> "快！"他吼道，"在他们赶来之前，砸碎这铁疙瘩！"

## 🎨 视觉降噪与反差设计（Visual Contrast Design）

### 核心原则

**"用绝对的安静与整洁，反衬统治阶级的残忍。"**

60万字的重金属感官会导致读者生理性审美疲劳。必须在重工业的血污中，穿插极度安静、精密、冰冷的场景。

### 两极对比

| 世界 | 感官基调 | 代表场景 |
|------|---------|---------|
| 🔴 **底层机工** | 震耳欲聋、煤烟、血污 | 天津卫铸造厂 |
| 🔵 **统治阶级** | 死寂、精密、冰冷 | 张廷远书房、奥古斯塔斯寝殿 |

### 执行规范

**底层机工场景**（朱靖渊/陈铁/顾铁兰 POV）:
- 必须描写：煤烟、机油、铁锈、轰鸣、血污、汗臭
- 感官：听觉（轰鸣）、嗅觉（煤烟）、触觉（烫伤）

**统治阶级场景**（张廷远/林霜降/奥古斯塔斯 POV）:
- 必须描写：死寂、精密、冰冷、一尘不染
- 感官：听觉（怀表滴答）、视觉（洁净）、触觉（冰冷黄铜）

### 执行示例

**底层场景**（第72章）:
> 铸造厂的空气里弥漫着煤烟和汗臭。几百个工人赤着上身，在蒸汽机的轰鸣声中挥汗如雨。铁锤砸在铁砧上，火星四溅。一个工人惨叫，他的手指被齿轮咬住了。

**统治阶级场景**（第45章）:
> 奥古斯塔斯女王的寝殿里，死寂。听不到外面的惨叫，只能听到纯金怀表秒针走动的"滴答"声。
>
> 她坐在轮椅上，双手颤抖。精致骨瓷杯里的红茶洒了一半在丝绸裙上，但她毫不在意。
>
> 她的眼神像一只老鹰，盯着窗外伦敦的煤烟。

## 💾 十章强制存档机制（Ten-Chapter Checkpoint）

### 核心原则

**"防止AI长线生成的'失忆综合征'。"**

面对220章、60万字的庞大生成量，任何AI大模型在写到第150章时，都极有可能发生"记忆模糊"（忘了林霜降的疤痕、忘了陈铁左腿的旧伤）。

### 规则

**每生成完10章，强制停下来，输出《前10章核心参数变动表》。**

### 存档内容

```json
{
  "checkpoint": "第11-20章",
  "timestamp": "天工三年春",
  "characters": {
    "陈铁": {
      "physical_state": "左腿有旧烫伤（第6章南洋自沉）",
      "location": "天津卫铸造厂",
      "relationships": {
        "老鬼": "师徒（第6章拜师）",
        "顾铁兰": "工友（第15章相识）"
      }
    },
    "林霜降": {
      "physical_state": "虎口有生铁烫伤疤痕（幼年）",
      "location": "张廷远府邸",
      "secret_activity": "秘密翻译《剥削考》"
    }
  },
  "items": {
    "铸铁大锤": {
      "holder": "老鬼",
      "history": "南洋自沉战用过的",
      "future_transfer": "第61章传给陈铁"
    }
  },
  "dead_characters": [
    "王铁胆（第2章大沽口炸膛）"
  ]
}
```

### 执行流程

```
1. 生成第11-20章
2. 暂停
3. 输出《核心参数变动表》
4. 更新 state.json
5. 检查：
   - 是否有死人复活？
   - 是否有断腿重生？
   - 是否有物品凭空消失？
6. 如果有问题，立即修复
7. 继续：生成第21-30章
```

## 🚫 禁止事项

### 绝对禁止

1. **禁止上帝视角** - 一章只能有一个POV
2. **禁止时间狂飙** - 必须遵守时间锁
3. **禁止死人复活** - 必须检查state.json
4. **禁止物品凭空消失** - 必须追踪物品去向
5. **禁止翻译腔** - 不用"不可思议"、"令人窒息"
6. **禁止排比句泛滥** - AI高潮时特别爱用三段式排比

### 警告信号

如果AI输出中出现以下信号，立即暂停并检查：

| 信号 | 问题 |
|------|------|
| "与此同时" | 可能违反POV锁 |
| "三天后" | 可能违反时间锁 |
| "不可思议" | 翻译腔 |
| "令人窒息" | 翻译腔 |

## ⛓️ 连贯性保证机制

### 黑匣子存档（每章末尾输出）

```
🩸 物理状态：[伤情、装备耐久、剩余物资]
📍 坐标位置：[角色精确地理环境]
ℹ️ 信息差：[谁掌握什么秘密]
🪝 悬念遗留：[未引爆的倒计时]
```

### 人物连贯性法则

| 法则 | 说明 |
|------|------|
| **伤痕焊死** | 一旦角色失去肢体，必须永久记录 |
| **精力消耗** | 高强度交锋后必须体现物理虚弱 |
| **性格底线** | 角色不能违背其核心利益 |

### 战力防膨胀法则

| 法则 | 说明 |
|------|------|
| **没有免费科技树** | 新武器必须有前置条件（经费、图纸、缺陷） |
| **质量守恒因果链** | 每个伏笔必须回收，每颗子弹必须有落点 |
| **敌我战力锚定** | 大明优势=本土决绝+生铁堆叠；洋人优势=精密+射程+组织 |

## 二、正确的《大明1900》开篇结构

### 方案一：陈铁视角（底层日常）

**场景**：天津卫铸造厂

**人物**：
- 陈铁（18岁机工学徒）
- 老鬼（炸酱面馆老板，前锅炉长）
- 阿牛（陈铁发小，即将被蒸汽烫死）

**流程**：
1. **庸俗日常**：陈铁在铸造厂打工，满脸煤黑，吃劣质窝头，牙齿咬到煤渣
2. **引入冲突**：洋人监工施罗德"不小心"让高压蒸汽烫死阿牛
3. **对话展开世界观**：老鬼用天津卫方言骂"这帮洋人，欺负到咱头上了"，引出八国联军逼宫的背景
4. **悬念钩子**：老鬼看着陈铁攥紧拳头，压低声音说："小陈，想报仇吗？今晚来我面馆，我有话说。"（暗示老鬼有秘密计划）

**语言细节**：
- 陈铁、老鬼：天津卫方言
- 施罗德：洋泾浜中文
- 内心独白：用方言思考

| **现代元素** | 手机、地铁、公园 | 电报、火车、私家园林 |
| **全知灌输** | "大明延续到1900年" | 通过对话逐步展开 |
| **概括叙述** | "天津卫很穷" | 陈铁咬到煤渣，牙齿发出脆响 |
| **语言同质** | 所有人都说官话 | 方言vs官话的阶层差异 |
| **满清元素** | 奴才、主子、辫子 | 臣、朕、发髻 |

---

## 四、核心原则

1. **具体场景**：不是概括，是画面
2. **人物性格**：通过对话展现，不是描述
3. **语言差异**：方言vs官话
4. **时代准确**：1900年大明，不是现代
5. **悬念钩子**：每章结尾必须有钩子

---

**核心**：学习《大明1937》的**结构骨架**，但用**1900年大明的真实元素**填充！

## 四、时代背景传递方式

### 方式一：电报新闻

**示例**：
```
电报员送来一份加急电报："兰芳省被英荷舰队封锁，
定远号自沉，全军覆没。"
```

**效果**：通过电报自然透露国际局势。

### 方式二：边缘思考

**示例**：
```
陈铁想到，自己从18岁进铸造厂，如今已经十年了。
十年前，天津卫还没有这么多洋人，也没有这么多蒸汽管道。
```

**效果**：通过人物回忆自然透露历史变迁。

### 方式三：环境细节

**示例**：
```
紫禁城的乾清宫里，巨大的蒸汽管道沿着墙角蜿蜒，
发出低沉的轰鸣声。朱靖镇坐在龙椅上，但龙椅下面
有轻微的震动——那是地下蒸汽泵在工作。
```

**效果**：不直接说"大明工业化改造了紫禁城"，而是通过细节展现。
