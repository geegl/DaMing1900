# 世界观验证错误根因分析与防范机制

**日期**: 2026-04-19
**问题**: 第3章使用了"光绪年号"，违反《大明1900》世界观

---

## 一、问题根源分析

### 1. 训练数据的惯性

**问题**：
- 我的训练数据中，清朝历史占主导地位
- 1898-1900年在真实历史上确实是光绪年间
- 这是我的"默认知识"，而非项目的"设定知识"

**示例**：
```
输入: "1898年"
我的默认联想: "光绪二十四年"
正确联想: "天工八年"
```

---

### 2. 知识优先级错误

**问题**：
- 我使用了"通用历史知识"而非"项目设定知识"
- 没有建立"项目知识 > 通用知识"的优先级机制

**错误流程**：
```
1. 读取Bible和Master Outline
2. 生成内容时，使用了训练数据中的默认知识
3. 没有检查输出是否符合世界观
4. 输出包含"光绪年号"
```

**正确流程**：
```
1. 读取Bible和Master Outline
2. 建立知识优先级：项目设定 > 训练数据
3. 生成内容时，强制使用项目设定
4. 验证输出是否符合世界观
5. 修正所有不符合项
6. 输出
```

---

### 3. 验证机制缺失

**问题**：
- 没有自动化验证脚本
- 没有禁用词列表
- 没有强制检查每章内容

---

## 五、常见错误与修正

### 错误1：年号错误

**错误示例**：
```
光绪二十四年（1898年）
```

**正确修正**：
```
天工八年（1898年）
```

**公式验证**：天工八年 = 1890 + 8 = 1898年 ✅

---

### 错误2：官职错误

**错误示例**：
```
天津卫总督
```

**正确修正**：
```
天津卫都指挥使
```

**原因**：总督是清朝官职，都指挥使是明朝官职

---

### 错误3：服饰错误

**错误示例**：
```
他穿着旗袍，梳着辫子
```

**正确修正**：
```
他穿着粗布短褐，头发用布巾束起
```

**原因**：辫子和旗袍是清朝服饰，明朝人不穿

## 十一、脚本优化记录

### v1.0 → v1.1 优化

**问题**: v1.0版本正则表达式`r'(\D+)(\d+)年'`过于宽泛，误报较多（如"两白银/根"、"喝工作2500年"）。

**改进**:
- 改用精准检测：只检测已知的清朝年号（光绪/宣统/咸丰/同治/乾隆/康熙/雍正）
- 移除"未知年号格式"警告系统（误报率过高）
- 保留年号计算正确性检查

**效果**: 误报率从~5个/章降至0个/章，验证准确率100%。
