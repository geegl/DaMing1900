## 一、公文体/宫廷风（庄重+压抑）

### 示例1：奏折体（张廷远风格）

```
臣廷远谨奏：

窃维今日之大明，正处于三千年未有之大变局。外有列强环伺，内有民心思变。兰芳一省，乃大明之血管，橡胶、生铁、原油，皆为战争之命脉。今英荷舰队封锁马六甲，实为断我大明之喉舌。

臣以为，当以退为进，先稳住洋人，暗中筹措军费。伦敦金融市场，英镑汇率波动，正值做空良机。臣已密令幕僚，暗中抛售英镑，筹集白银三百万两，以备军需。

此计不可为外人道，尤其是陛下身边之人。臣深夜密奏，望陛下三思。

臣廷远顿首。
天工十年三月十五日
```

**毛边特征**：
- 半文半白
- 夹杂西方词汇（英镑、汇率）
- 语重心长但暗藏算计

### 示例2：圣旨体（朱靖镇风格）

```
奉天承运，皇帝诏曰：

朕闻古之立国者，必有重器。今西洋诸国，以铁甲舰称雄海疆，以蒸汽机横行大陆。我大明立国二百五十六年，岂能甘居人后？

着令工部、兵部、户部，合力督办铸造厂事宜。所需银两，由内帑拨付。所需铁料，由兰芳省调运。所需工匠，由各地招募。

朕不要借口，朕要铁甲舰。三年之内，朕要看到十艘铁甲舰下水。

钦此。

天工元年正月初一
```

**毛边特征**：
- 极短句
- 命令式
- 重金属压迫感

## 二、说书味/市井风（粗糙+生动）

### 示例3：天津卫方言（老鬼风格）

```
老鬼用那只粗糙的铸铁义肢，在桌上一磕，发出"咣当"一声响。

"他娘的，这世道！"老鬼骂道，"洋人来咱天津卫，那是客。可这客，他娘的骑到咱头上了！"

陈铁蹲在墙角，没说话。

老鬼继续骂："你看看那帮监工，吃的是白面馒头，喝的是洋酒。咱呢？咱吃的是煤灰拌饭，喝的是凉水！这叫嘛事？这叫欺负人！"

老鬼吐了一口唾沫，唾沫里带着黑痰。

"小陈，你记住。"老鬼压低声音，"这世道，不能认命。咱得憋着，等机会。"

陈铁抬起头，看着老鬼。

"啥机会？"

老鬼嘿嘿一笑，露出一口被烟熏黄的牙齿。

"到时候你就知道了。"
```

**毛边特征**：
- 天津卫方言（"嘛事"、"他娘的"）
- 动作描写（铸铁义肢磕桌、吐黑痰）
- 断句、重复

### 示例4：市井对话（《金瓶梅》风格）

```
"掌柜的，来碗炸酱面。"

"好嘞，您了稍等。"

老鬼转身去灶台，用一只手擀面条。那面条在他手里转来转去，不一会儿就薄如蝉翼。

"您了看，这面条怎么样？"

"行，够筋道。"

老鬼把面条扔进锅里，开水翻滚，面条在水中翻腾。

"您了是铸造厂的？"

"对。"

"那厂子，最近怎么样？"

"还能怎么样，还不是那样。洋人监工，天天骂人。"

老鬼盛起面条，浇上炸酱，端到桌上。

"您了慢慢吃。有嘛事，尽管说。"
```

**毛边特征**：
- 天津卫方言（"您了"）
- 动作细节（擀面条、扔进锅里）
- 简短对话，不绕弯子

## 三、蒸汽反差/工业风（视觉冲击）

### 示例5：蒸汽管道轰鸣

```
蒸汽管道沿着墙壁蜿蜒，像一条黑色的巨蟒。管道外壁上满是油污和煤灰，有些地方包着粗糙的麻布，用来隔热。

管道里发出低沉的轰鸣声，震得陈铁的耳朵嗡嗡响。他伸手摸了摸管道的外壁——烫。但他没缩手，继续摸着，感受温度的分布。

煤烟从厂房顶棚的缝隙里飘下来，落在米饭上。陈铁没有挑出来。他用筷子搅了搅，把煤灰和米饭一起扒进嘴里。

牙齿咬到一粒硬的东西，发出"咯嘣"一声脆响。是煤渣。

他嚼了两下，咽下去。
```

**毛边特征**：
- 具体感官细节（烫、煤渣、咯嘣声）
- 粗粝感（油污、煤灰、麻布）
- 动作描写，不形容词

### 示例6：铁甲舰自沉

```
定远号的巨大铁甲舰，缓缓沉入海底。

煤烟遮天，蒸汽管道爆裂，发出刺耳的嘶嘶声。水兵们跳进海里，扑腾着向岸边游去。

老鬼站在甲板上，用一只手握着扳手。他的左臂已经被蒸汽烫得血肉模糊，但他没感觉。

"炸了。"老鬼低声说。

他转身，走向锅炉舱。那里，有最后一根蒸汽燃爆管。

"老鬼！你疯了！"有人在后面喊。

他走进锅炉舱，点燃了燃爆管。

"轰——"
```

## 📋 随机抽取规则

**Python代码示例**：

```python
import random

def inject_reference_texts():
    with open("docs/03-参考层/Reference_Text_Library.md", "r", encoding="utf-8") as f:
        content = f.read()

    # 提取所有示例
    examples = content.split("### 示例")[1:]  # 去掉第一个空元素

    # 分类
    public_style = []      # 公文体/宫廷风
    story_style = []       # 说书味/市井风
    industrial_style = []  # 蒸汽反差/工业风

    for example in examples:
        if "公文体" in example or "宫廷风" in example:
            public_style.append(example)
        elif "说书味" in example or "市井风" in example:
            story_style.append(example)
        else:
            industrial_style.append(example)

    # 随机抽取
    selected = [
        random.choice(public_style),
        random.choice(story_style),
        random.choice(industrial_style)
    ]

    return "\n\n---\n\n".join(selected)
```
