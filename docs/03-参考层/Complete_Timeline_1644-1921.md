### 天工帝朱靖镇（关键人物）

**基本信息**：
- **庙号**：熹宗
- **年号**：天工
- **在位**：1896-1909（13年）
- **谥号**：烈皇帝
- **出生**：1871年
- **登基**：1896年（25岁）
- **驾崩**：1909年（38岁）

**登基背景**：
- 1896年，光绪帝驾崩，无子嗣
- 从宗室中选择朱靖镇（宪皇帝次子）
- 朱靖镇沉迷军工，被朝臣视为"圣君"

**在位大事**：
- 1898年：颁布《限铁令》，铁本位开始
- 1900年：大沽口失守（第2章）
- 1905年：定远号自沉（第5-6章）
- 1909年：重金属中毒驾崩，传位朱靖渊

## 四、第一次世界大战（1914-1918）

> **架空世界名称**：欧陆大战 / 西方大战

### 战争性质

**真实历史**：第一次世界大战
**架空名称**：**欧陆大战**（1914-1918）

**大明称呼**：西方大战 / 泰西大战

---

### 战争背景

**同盟国**：
- 德国
- 奥匈帝国
- 奥斯曼帝国

**协约国**：
- 英国
- 法国
- 俄国
- 美国（1917加入）

**大明立场**：中立（1914-1917）→ 站队协约国（1917-1918）

---

### 关键时间节点

| 日期 | 事件 | 对大明影响 |
|------|------|----------|
| **1914-06-28** | 萨拉热窝事件 | 无直接影响 |
| **1914-07-28** | 奥匈对塞尔维亚宣战 | 大明宣布中立 |
| **1914-08** | 德国入侵比利时 | 英国参战 |
| **1915** | 日本对德宣战 | 大明警惕日本 |
| **1916** | 凡尔登战役/索姆河战役 | 欧洲消耗战 |
| **1917-02** | 德国无限制潜艇战 | 大明商船被击沉 |
| **1917-03** | 俄国二月革命 | 沙俄崩溃，北方威胁解除 |
| **1917-04** | 美国参战 | 大明转向协约国 |
| **1917-11** | 俄国十月革命 | 苏俄退出战争 |
| **1918-03** | 德国西线进攻 | 最后挣扎 |
| **1918-11-11** | 停战协定 | 大明作为战胜国 |

---

### 大明参战过程

| 时间 | 事件 | 章节 |
|------|------|------|
| **1914-1916** | 中立，向双方出售军火 | 第116-120章 |
| **1917-02** | 德国潜艇击沉大明商船 | 第126章 |
| **1917-04** | 大明对德宣战 | 第128章 |
| **1917-06** | 大明劳工赴欧洲 | 第130章 |
| **1918-11** | 大明参加巴黎和会 | 第145章 |

---

### 对大明的影响

**正面**：
- 作为战胜国，国际地位提升
- 英国被迫归还部分兰芳权益（第145章后）
- 德国技术援助（战后）

**负面**：
- 日本趁机扩大在朝鲜影响力
- 战争期间工业消耗巨大
- 战后西方列强恢复，重新压迫大明

---

### 与泰晤士河口威慑战的关系

**时间**：1918年（第130-145章）

**背景**：
- 欧陆大战进入最后阶段
- 德国在西线挣扎
- 英国本土防守空虚

**洪威帝决策**：
- 趁英国无力东顾，派遣舰队威慑泰晤士河口
- **这是压垮德国的最后一根稻草**（英国被迫两线作战）
- 洪威帝成为"决定欧陆大战走向的人"
