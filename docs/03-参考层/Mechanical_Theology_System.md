## 三、第一层：技术神学化

### 《天工开物·蒸汽卷》典型口诀

**卷一：锅炉操作口诀**

---

**【坎水化气诀】**

```
坎水入离炉，火旺生白气。
气满则冲关，气压须慎记。
    
坎一离九数，压力随火起。
白气冲天际，炉膛不可欺。
    
若见气发黑，火衰即停机。
急开坎水阀，降温保太平。
    
切忌气满溢，爆裂毁天机。
天工有明训，生死在呼吸。
```

**真实物理原理**：

| 口诀 | 物理原理 |
|------|---------|
| **坎水入离炉** | 水注入锅炉（坎=水，离=火）|
| **火旺生白气** | 高温产生蒸汽 |
| **气满则冲关** | 压力升高→打开安全阀 |
| **压力随火起** | 温度↑→压力↑（PV=nRT）|
| **气发黑=火衰** | 蒸汽变黑=燃烧不充分 |
| **气满溢=爆裂** | 压力过高→锅炉爆炸 |

**关键**：机工只知道"背口诀"，不知道"PV=nRT"

---

**【锅炉压力分级口诀】**

```
一气轻如丝，压力安如山。（压力<5 atm）
二气白如雾，运转保平安。（压力5-10 atm）
三气浓如云，马力翻一番。（压力10-15 atm）
四气黑如墨，危险在眼前。（压力15-20 atm，临界）
五气红如血，爆裂毁人寰。（压力>20 atm，必炸）
```

**压力与颜色对应表**：

| 压力范围 | 蒸汽颜色 | 锅炉状态 | 操作 |
|---------|---------|---------|------|
| **<5 atm** | 几乎看不见（轻如丝）| 安全 | 正常运转 |
| **5-10 atm** | 白色雾状 | 正常 | 正常运转 |
| **10-15 atm** | 浓白色云状 | 高效 | 提高效率 |
| **15-20 atm** | 灰黑色（黑如墨）| 危险 | **立即降温** |
| **>20 atm** | 红色（红如血）| 必炸 | **紧急撤离** |

**关键**：用颜色判断压力，无需压力表（底层机工看不懂）

**卷二：差分机操作口诀**

---

**【齿轮咬合诀】**

```
乾三连坤六断，齿轮相咬合。
三转六同步，计算无差错。
    
一乾生二坤，进位在指尖。
咬合须精准，公差不可偏。
    
若闻咔咔声，齿轮已磨损。
急停更旧件，否则毁天机。
    
切记坤不转，乾必随停机。
连锁反应起，整阵毁一旦。
```

**真实机械原理**：

| 口诀 | 机械原理 |
|------|---------|
| **乾三连坤六断** | 齿轮比3:6=1:2 |
| **三转六同步** | 小齿轮转3圈，大齿轮转6圈 |
| **一乾生二坤** | 一级齿轮带动二级齿轮 |
| **咬合须精准** | 齿轮啮合精度要求高 |
| **公差不可偏** | 公差<0.01mm |
| **咔咔声=磨损** | 异响=齿轮磨损 |
| **坤不转=乾停** | 从动轮卡死→主动轮停 |

**卷三：蒸汽管道维修口诀**

---

**【管道泄压诀】**

```
蒸汽如龙行，管道是其躯。
龙怒则冲破，泄压在须臾。
    
龙首接锅炉，龙尾通机器。
中途若堵塞，龙怒毁屋壁。
    
若闻龙啸声，压力已危急。
急转泄压阀，龙息得安息。
    
切记顺时针，龙怒不可欺。
逆时针旋转，龙息冲天际。
```

**真实物理原理**：

| 口诀 | 物理原理 |
|------|---------|
| **蒸汽如龙行** | 蒸汽在管道中流动 |
| **龙怒=冲破** | 压力过高→管道爆裂 |
| **泄压在须臾** | 必须快速降压 |
| **龙啸声=危急** | 高压蒸汽泄漏的尖啸声 |
| **泄压阀** | 安全阀 |
| **顺时针=泄压** | 顺时针转动阀门→泄压 |
| **逆时针=冲天际** | 逆时针转动→压力更高→爆炸 |

## 四、第二层：符箓式图纸解读规范

### 图纸绘制规则

**传统图纸**（西方）：
- 透视图、三视图
- 标注尺寸、公差
- 比例尺
- 物理单位（mm、atm）

**大明符箓图纸**：

| 特征 | 内容 | 作用 |
|------|------|------|
| **无透视** | 平面绘制，无立体感 | 机工无法理解结构 |
| **符文标注** | 用《易经》卦象代替数字 | 机工看不懂真实尺寸 |
| **颜色编码** | 红=危险，绿=安全，黑=正常 | 直观判断，无需理解 |
| **禁忌符号** | 画骷髅=禁忌部位，画龙=高压区 | 警示，无需解释 |

---

**【锅炉压力检测点符箓】**


**解读口诀**：

```

```

**机工操作**：
- 看到**红色离卦** → 知道高温，远离
- 看到**蓝色坎水** → 知道冷水入口
- 看到**黄色龙纹** → 知道蒸汽区，小心
- 看到**白色白气** → 知道安全区
- 看到**黑色骷髅** → 知道禁忌，绝不可碰

**关键**：
- 机工**不需要理解**锅炉结构
- 只需要**识别颜色和符号**
- 按照口诀操作即可

## 六、林霜降的"蒸汽宗教"传播

### 《剥削考》的伪装策略

**原文**（马克思主义）：

```
资本家剥削工人的剩余价值。
工人创造的价值，大部分被资本家占有。
工人只得到维持生存的最低工资。
```

**伪装后**（蒸汽宗教）：

```
【蒸汽造化经·卷一】

坎水化气，气推机轮。
机轮转动，生出万金。
    
坎水者，工也。
机轮者，器也。
万金者，利也。
    
水流进炉，气推机轮。
机轮运转，生出黄金。
    
然黄金何在？
    
不在坎水，不在机轮。
而在掌炉之人。
    
坎水付出身，机轮转动骨。
掌炉收黄金，坎水得煤灰。
    
此乃天道不公也。
    
天道不公，当如何？
    
断其蒸汽，毁其机轮。
复归自然，各得其所。
```

---

### 核心概念的伪装对照

| 马克思主义概念 | 蒸汽宗教术语 |
|--------------|-------------|
| **工人阶级** | **坎水**（水流进锅炉，象征工人付出）|
| **生产资料** | **机轮**（机器设备）|
| **剩余价值** | **黄金**（机轮运转产生）|
| **资本家** | **掌炉人**（控制锅炉的人）|
| **剥削** | **坎水得煤灰**（工人只得到煤灰，黄金被掌炉人拿走）|
| **革命** | **断其蒸汽，毁其机轮**（停止生产，推翻剥削）|
| **阶级斗争** | **天道不公，当如何**（质疑不公）|

---

### 机工的理解

**文盲机工的解读**：

```
我是坎水，流进锅炉。
锅炉是机轮，我推动它。
机轮生出黄金，但我只得煤灰。
    
掌炉人（张廷远、皇帝）拿走黄金。
    
这不公平！
    
所以，要断蒸汽，毁机轮！
```

**关键**：
- 机工**不懂**"剩余价值"、"剥削"等术语
- 但**看得懂**"坎水得煤灰，掌炉收黄金"
- 用蒸汽隐喻解释马克思主义
