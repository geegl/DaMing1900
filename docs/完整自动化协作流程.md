## 三、用户控制方式

### 方式1：直接对话（推荐）

```
你：开始写，不要停，一直写下去
我：立即开始写作，每章Codex审查，每10章发Telegram报告

你：写到第50章停下
我：写完第50章后停止，发送总结报告

你：继续写
我：从第51章继续
```

### 方式2：Telegram命令（后台运行）

```bash
# 启动命令监听器
python3 automation/scripts/telegram_command_listener.py &
```

然后通过Telegram发送命令：

| 命令 | 功能 |
|------|------|
| `/status` | 查看当前进度 |
| `/next 10` | 写10章 |
| `/pause` | 暂停 |
| `/resume` | 恢复 |
| `/review 15` | Review第15章 |
| `/fix 第3章时间线错误` | 修正问题 |

### 方式3：I-Lang自动化

```bash
# 运行I-Lang技能
python3 automation/scripts/ilang_runner.py --skill chapter_gen.ilang --chapter 004-010
```

## 五、自动化流程伪代码

```python
def auto_write(target_chapters=220):
    """自动化写作主循环"""
    
    # Phase 0: 上下文预加载
    cache_context()
    
    current_chapter = get_current_chapter()
    
    while current_chapter <= target_chapters:
        
        # 检查暂停命令
        if check_pause_command():
            send_telegram("⏸️ 已暂停写作")
            break
        
        # Phase 1: 8-Agent写作
        chapter_content = run_8_agents(current_chapter)
        
        # Phase 2: Codex审查
        review_result = codex_adversarial_review(chapter_content)
        
        # 未通过则修正
        while review_result.score < 48:
            chapter_content = apply_fixes(chapter_content, review_result.issues)
            review_result = codex_adversarial_review(chapter_content)
        
        # Phase 3: 世界观验证
        worldview_result = run_worldview_validator(chapter_content)
        if not worldview_result.passed:
            chapter_content = fix_worldview_issues(chapter_content, worldview_result.issues)
        
        # Phase 4: 保存章节
        save_chapter(current_chapter, chapter_content)
        
        # Phase 5: 状态同步
        update_state_json(current_chapter)
        git_commit(f"完成第{current_chapter}章")
        git_push()
        
        # Phase 6: Telegram通知
        send_chapter_complete_notification(current_chapter, review_result.score)
        
        # 批次Review（每10章）
        if current_chapter % 10 == 0:
            batch_review(current_chapter // 10)
            send_batch_complete_notification(current_chapter // 10)
        
        # 中期Review（每50章）
        if current_chapter % 50 == 0:
            midterm_review(current_chapter)
            send_midterm_review_notification(current_chapter)
        
        # 阶段Review（每部）
        if current_chapter in [70, 165, 220]:
            stage_review(get_part(current_chapter))
            send_stage_review_notification(get_part(current_chapter))
        
        # 下一章
        current_chapter += 1
    
    # 完成
    send_telegram(f"🎉 写作完成！共{target_chapters}章")
```
