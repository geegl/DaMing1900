**但仍缺失**：
- 铁币如何流通？
- 大宗交易如何进行？
- 官方兑换机制

---

### 完整修正方案

#### 1. 铁币流通机制

**规则**：
- 铁币印有"铸造月份"
- 1-6个月：全额流通
- 7-12个月：按月折价（第7个月=90%，第12个月=50%）
- 超过12个月：拒绝流通，必须去官府兑换

**官府兑换窗口**：
- 11个月的旧币可兑换0.5个新币（铸币税50%）
- 12个月以上的铁粉强制回收，不给兑换

**效果**：
- 底层被迫每月花光铁币（促进消费，锁死储蓄）
- 官府收取50%铸币税（剥削再加一层）

---

#### 2. 大宗交易机制

**问题**：铁币不适合大宗交易（太重、会氧化）

**解决方案**：
- **官方银票**：用于大宗交易（如购买机器、原材料）
- **铁币**：仅用于底层日常交易（工资、口粮）

**双轨制**：
| 交易类型 | 货币 | 适用人群 |
|---------|------|---------|
| **大宗交易** | 银票（不氧化）| 买办、工厂主 |
| **日常交易** | 铁币（氧化）| 底层机工 |
| **黑市交易** | 实物（润滑油、废铁）| 起义军 |

## 四、人机复合算阵"I/O瓶颈"修正 ⚠️ 新发现

### 审计者质疑

**问题**：10万人如何与差分机交互？

**要求**：设计数据传递机制（气动传声管或光学投影）。

---

### 修正方案

#### 地下算阵的物理架构

```
紫禁城地下30米

┌─────────────────────────────────────┐
│  差分机核心区（500台精密机器）        │
│  - 纯机械计算                        │
│  - 误差<1%自动输出                   │
└───────────┬─────────────────────────┘
            │ 气动传声管
            ↓
┌─────────────────────────────────────┐
│  光学投影大厅                        │
│  - 巨大透镜将差分机输出投影到墙面    │
│  - 100个奴工同时读取                 │
└───────────┬─────────────────────────┘
            │ 纸带传递
            ↓
┌─────────────────────────────────────┐
│  珠算奴工区（10万人）                │
│  - 读取纸带数据                      │
│  - 用算盘验算                        │
│  - 通过传声管汇报误差                │
└─────────────────────────────────────┘
```

#### 数据传递机制

**1. 差分机→奴工（下行）**
- 差分机输出打孔纸带
- 纸带通过气动管道传输到奴工区
- 100个奴工同时读取纸带

**2. 奴工→差分机（上行）**
- 奴工发现误差>5%
- 通过传声管报告（铜管+振动膜）
- 差分机操作员听到后手动调整

**速度**：
- 纸带传输：5秒
- 奴工验算：20秒
- 汇报反馈：5秒
- **总计：30秒**
