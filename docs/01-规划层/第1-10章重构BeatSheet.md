## 第1章《煤烟落在米饭上》

### 核心场景
天津卫铸造厂，午时。陈铁在铸铁边角料堆上吃冷饭，遭遇工籍烙印检查。

### 感官压抑元素（≥3处）
1. **铁币生锈**: 陈铁掏出两个铜板买馒头，铜板边缘有暗红色锈迹（铁本位货币，铁锈味混入食物）
2. **工籍烙印**: 工头检查工籍牌，陈铁展示左臂烙印——"津铸丙字七十四号"，烙印边缘发红，渗着血丝（新烙3个月）
3. **煤烟天空**: 煤灰簌簌落在米饭上，陈铁没挑，直接嚼，牙齿咬到煤渣发出脆响

### 制度压抑元素（≥1处）
- **工籍检查**: 工头带着锦衣卫校尉铁头突然检查工籍，没有工籍牌的人被当场拖走（后文铺垫阿牛的恐惧）
- **配给制度**: 工人每人每天只有2个铜板餐补，冷饭是前一天的，硬得像石头

### 政治暗线（无，但铺垫）
- 张大人（张廷远）视察工厂，带着洋人施罗德
- 林霜降首次出场（白丝绒手套，丝绸长裙），观察陈铁的扳手

### 关键对话修改
```
工头:"工籍牌！都掏出来！"
陈铁撸起袖子，露出手臂上的烙印。铁制的工籍牌挂在腰间，上面印着:
"天津卫军械铸造厂
  丙字第七十四号
  光绪二十六年正月入厂"
铁头（锦衣卫校尉）走过来，看了一眼陈铁的烙印:"新入厂的？"
"三个月。"
"烙印发红，没化脓，算你走运。"铁头冷笑，"上一批有个小子，烙印感染了，整个胳膊都烂了，扔乱葬岗了。"
```

### 视觉锚定
- 陈铁的工籍烙印:左臂内侧，长方形，边缘红肿，渗着黄色组织液
- 铁币:暗红色，边缘磨损，有铁锈味
- 煤烟落在米饭上:米粒是黄的，煤灰是黑的，混在一起像"撒了胡椒的冷粥"

## 第2章《大沽口的劣质炮》

### 核心场景
老鬼的炸酱面馆，午时。逃兵带来大沽口失守的消息。

### 感官压抑元素（≥3处）
1. **煤烟落在面上**: 煤烟从窗缝钻进来，落在面团上，老鬼没挑，直接和面
2. **生铁义肢**: 老鬼的五斤重生铁义肢撞在案板上，发出闷响，义肢接缝处渗着黑油
3. **逃兵的冻伤**: 逃兵右脚脚趾紫黑，烂了三个，散发着腐肉味

### 制度压抑元素（≥1处）
- **宵禁**: 逃兵提到大沽口已实行宵禁，晚上六点后人不得出门，违者枪毙
- **连坐**: 炮台失守后，守将全家被下诏狱，士兵逃亡者株连三族

### 政治暗线（无，但铺垫）
- 逃兵提到晋商钱四海负责军需采购
- **北方威胁暗示**: 逃兵提到"北方商人"给钱四海送过东西

### 关键对话修改
```
逃兵:"炮管自己炸了，把王将军的腿炸断了。那铁料……不对劲。"
老鬼:"什么不对劲？"
逃兵:"铁料是北方来的。晋商钱四海，背后有人。"
"谁？"
"不知道。"逃兵摇头，压低声音，"但有人说，在天津卫见过北方商人。穿皮毛的，说话带口音，不是咱们这儿的调。他们给钱四海送过东西，装在木箱里，沉甸甸的。"
老鬼的手停了一瞬:"什么口音？"
"说不上。硬邦邦的，像刀子刮铁。"
```

### 视觉锚定
- 老鬼的义肢:生铁铸，表面粗糙，铆钉固定，接缝处渗黑油
- 逃兵的脚:棉裤上暗红一片（血），脚趾紫黑腐烂
- 煤烟:从窗缝钻进来，像"给死人盖被子"

## 第3章《阿牛之死》

### 核心场景
铸造厂三号厂房，申时。阿牛因疲劳驾驶蒸汽机甲，被施罗德殴打致死。

### 感官压抑元素（≥3处）
1. **工籍烙印发炎**: 阿牛展示工籍烙印（丙字七十六号），烙印边缘化脓，散发着腐臭味
2. **煤烟与血**: 阿牛被打后吐血，血混着煤烟落在地上，变成黑色的泥浆
3. **蒸汽 burns**: 阿牛被施罗德的机甲喷出的蒸汽烫伤，皮肤起了水泡，散发着熟肉味

### 制度压抑元素（≥1处）
- **工籍检查**: 施罗德检查阿牛的工籍牌，发现烙印模糊（其实是故意找茬）
- **连坐**: 阿牛死后，他的工牌被收走，工头说:"家里的人连坐，三个月不许进厂"

### 政治暗线（无）
- 钱四海的手下在厂外接应施罗德

### 关键修改:尸体处理
```
阿牛死了。
施罗德的皮靴踩在阿牛胸口，像是在踩一块破布。
"拖走。"施罗德说。
钱四海的手下跑过来，两个人架起阿牛的尸体。
"扔哪儿？"其中一个问。
"乱葬岗。"钱四海的手下说，"没工籍的，不配入土。"
陈铁抬起头:"他有工籍。"
"工籍？"那人冷笑，举起阿牛的工牌，突然用力一折，铁质的工牌断了，"现在没了。"
他把断成两半的工牌扔在阿牛尸体上。
"丙字七十六号，阿牛，病亡。"他掏出个小本本，在上面写了几个字，"家属连坐，三月不得入厂。"
```

### 视觉锚定
- 阿牛的工籍烙印:丙字七十六号，边缘化脓，皮肤发红溃烂
- 尸体:被拖走时，手在铸铁地面上划出五道血痕
- 断成两半的工牌:铁质的，断口发白，扔在尸体胸口

## 第4章《老鬼的秘密集会》

### 核心场景
老鬼的炸酱面馆，深夜。地下情报网第一次集会。

### 感官压抑元素（≥3处）
1. **铁币生锈**: 老鬼收取会费（每人5个铁币），铁币在桌上堆成一小堆，散发着铁锈味
2. **生铁义肢漏油**: 老鬼的义肢在阴雨天疼痛，接缝处渗着黑油和血水的混合物
3. **煤烟与黑暗**: 面馆里没有点灯，只有煤炉的微光，煤烟在黑暗中像幽灵一样飘动

### 制度压抑元素（≥1处）
- **宵禁**: 集会在宵禁后进行，外面有锦衣卫巡逻，大家屏住呼吸
- **工籍检查**: 入会必须查验工籍烙印，确认是"自己人"

### 政治暗线（重要）
- **小德子出场**: 面馆跑堂，15岁，眼神锐利，动作麻利
- 小德子其实是司礼监刘谨言派来的联络人

### 关键场景:小德子出场
```
门帘一挑，进来一个跑堂的小子。
十五六岁的模样，眉眼清秀，面白无须，嗓音尖细。眼神不对——太锐了，像刀子。
"老板，面好了。"他把一碗面放在老鬼面前，动作麻利，但右手食指上有茧——握笔的茧，不是干活的茧。
老鬼看了他一眼:"新来的？"
"嗯，昨天来的。叫小德子。"他低着头，但眼睛在扫视屋子里的人，"老板，需要加汤吗？"
"不用。"老鬼说，"去门口看着，有动静敲三下锅。"
"明白。"小德子转身出去，脚步轻得像猫。
陈铁盯着他的背影:"这人不对。没胡子，嗓子也细……"
"嗯。"老鬼喝了口面汤，"宫里出来的。上面有人的。"
"什么人？"
老鬼没回答，只是用义肢敲了敲桌面:"能用就行。"
```

### 视觉锚定
- 小德子:15岁，眉眼清秀，眼神锐利，右手食指有握笔的茧
- 铁币:5个铁币在桌上，暗红色，边缘生锈
- 义肢渗液:黑油+血水，在桌面上留下污渍

## 第5章《定远号的铁棺材（上）》

### 核心场景
老鬼的回忆:1894年，定远号自沉之夜。

### 感官压抑元素（≥3处）
1. **海水与铁锈**: 定远号的铁甲生锈，海水灌进来，铁锈味混着咸腥味
2. **断臂之痛**: 老鬼炸毁锅炉时被碎片削断左臂，骨头断裂的声音，血喷在生铁甲板上
3. **蒸汽 burns**: 锅炉爆炸的蒸汽烫伤了老鬼的右半边脸，皮肤起水泡，散发着熟肉味

### 制度压抑元素（≥1处）
- **军籍注销**: 老鬼断臂后，军籍被注销，工部给的抚恤金只有5个铁币（讽刺）
- **伤残制度**: 伤残士兵不得留营，一律遣散，自生自灭

### 政治暗线（重要）
- **老鬼暗示"我有上线"**
- 揭示老鬼与东厂探事孙猴子的关系
- 老鬼提到"宫里的人也在查钱四海"

### 关键对话修改
```
陈铁:"你为什么要查钱四海？"
老鬼沉默了很久，义肢在桌面上轻轻敲击。
"我有上线。"老鬼说。
"什么？"
"宫里的人。"老鬼的声音很低，"司礼监。他们在查钱四海，查大沽口的炮管。但他们不能露面，需要底下的人。"
陈铁:"你信他们？"
"不信。"老鬼笑了，那笑容比哭还难看，"但敌人的敌人，就是朋友。"
```

### 视觉锚定
- 定远号的铁甲:生锈的铆钉，被海水浸蚀的钢板
- 断臂:骨头白森森的，血喷在甲板上，和铁锈混在一起
- 老鬼年轻时的脸:右半边被蒸汽烫伤，皮肤皱缩

## 第6章《定远号的铁棺材（下）》

### 核心场景
回到现实，老鬼的面馆。铁本位货币的剥削细节。

### 感官压抑元素（≥3处）
1. **铁币生锈**: 陈铁数自己的积蓄——37个铁币，每个都生锈，边缘磨损，散发着铁锈味
2. **工籍牌冰冷**: 陈铁的工籍牌挂在腰间，铁质，冬天冷得像冰，贴着皮肤让人发抖
3. **煤烟与饥饿**: 陈铁一天没吃饭，胃里空得发慌，煤烟味让他更加恶心

### 制度压抑元素（≥1处）
- **铁本位剥削**: 工人发工资用铁币，铁币容易生锈，贬值快，买办用铜钱兑换铁币时压价
- **配给削减**: 朝廷以"大沽口战败"为由，削减工人配给，每天少发一个馒头

### 政治暗线（无）
- 钱四海派人监视老鬼的面馆

### 关键修改:铁本位剥削
```
陈铁数了数手里的铁币。
三十七个。每个都沉甸甸的，边缘生锈，散发着铁腥味。
这是他的全部积蓄。
铁币是大明工籍专用的配给货币。银两才是硬通货，能买洋货、坐洋船、去南洋。但机工没有银两——工资发的是铁币，只能在官办铺子里买配给的口粮。
铁币会生锈。生锈的铁币，重量减轻，价值贬低。钱四海的兑换铺子，用铜钱收生锈的铁币，十个生锈的铁币换八个铜钱，再拿铜钱去换新的铁币，中间赚两成。
这就是剥削。
陈铁的铁币，已经生了锈。暗红色的锈迹，像血一样。
他想去看海。看海需要银两——船票要银子，铁币不收。但陈铁没有银子，只有这些生锈的铁币。
还差十三个铁币，才能换成足够的铜钱，再换成碎银。
陈铁把铁币装回布袋，贴身收好。铁币冰冷，贴着皮肤，像一把刀。
```

### 视觉锚定
- 37个铁币:暗红色，边缘生锈，堆在一起像一小堆凝固的血块
- 工籍牌:铁质，冰冷，冬天贴着皮肤让人发抖
- 钱四海的兑换铺子:门口的牌子写着"公平交易，童叟无欺"，讽刺

## 第7章《兰芳的血》

### 核心场景
陈铁从老鬼处听说兰芳橡胶林被烧，表哥生死不明。

### 感官压抑元素（≥3处）
1. **铁币的冰冷**: 陈铁想寄钱给兰芳的表哥，但铁币太重，寄不走（南洋不收生锈的铁币）
2. **信纸的焦黄**: 老鬼拿出表哥的信，信纸被煤烟熏得焦黄，边缘发黑
3. **血腥的描述**: 老鬼讲述兰芳血战，橡胶林被烧，工人被屠杀，血把泥土都染红了

### 制度压抑元素（≥1处）
- **邮政审查**: 往南洋的信件必须经工部检查，任何提及"军情"的内容都会被扣下
- **连坐**: 兰芳的工人被屠杀后，家属在大明被株连，不许入厂，不许领抚恤

### 政治暗线（北方威胁）
- 老鬼提到"北方的皮毛商"也在倒卖军火给洋人
- 暗示满清残部在兰芳事件中的角色

### 关键修改:北方势力暗示
```
老鬼:"兰芳的事，不只是洋人。"
陈铁:"什么意思？"
"有人在背后给洋人递刀。"老鬼的声音很低，"北方的皮毛商，你们见过的那种——穿皮袍子，说话带口音。他们在兰芳活动，卖给洋人的不只是军火，还有地图。工人的营地图，橡胶林的布防图。"
"他们是谁？"
"不知道。"老鬼摇头，"但他们在天津卫也有据点。钱四海跟他们有来往。"
陈铁想起第2章逃兵说的话:"给钱四海送东西的人？"
"嗯。"老鬼的义肢敲了敲桌面，"这水，比你想的深。"
```

### 视觉锚定
- 表哥的信:信纸焦黄，边缘被煤烟熏黑，字迹潦草
- 兰芳的橡胶林:被烧后黑漆漆的，像一片焦土
- 北方的皮毛商:穿皮袍子，腰间挂着沙俄制造的左轮枪

## 第8章《炮管里的砂眼》

### 核心场景
陈铁潜入铸造厂废料堆，发现劣质炮管的秘密。

### 感官压抑元素（≥3处）
1. **铁锈的味道**: 废料堆里全是生锈的铁块，散发着浓重的铁腥味
2. **砂眼的触感**: 陈铁摸到炮管内壁的砂眼，粗糙得像砂纸，能刮破手指
3. **机油的黏腻**: 陈铁的手上沾满黑机油，黏腻腻的，擦不干净

### 制度压抑元素（≥1处）
- **工籍限制**: 工人不得进入废料堆（陈铁是偷溜进去的）
- **技术保密**: 任何讨论炮管质量的人都会被锦衣卫带走

### 政治暗线（无）
- 施罗德出现，与钱四海密谈

### 关键对话修改:施罗德的证词
```
施罗德看着炮管，用手指摸了摸内壁的砂眼。
他的眉头皱了起来。
"这铁料……"他用德语说了一句什么，然后切换成中文，"德国人也不会用这种铁料。"
陈铁躲在废料堆后面，屏住呼吸。
"这是你们大明的铁。"钱四海的声音带着笑，"德国人的标准太高，我们用不起。"
"这不是标准的问题。"施罗德的声音冷了下来，"这是谋杀。用这种炮管，炮手必死无疑。"
"那就让炮手小心点。"钱四海说。
施罗德沉默了很久。然后他转身离开，靴子踩在铁屑上，发出咯吱咯吱的声响。
"我会报告领事馆。"他说，"但不会有人在乎。"
```

### 视觉锚定
- 炮管内壁的砂眼:粗糙，像蜂窝一样，能塞进指甲
- 废料堆:生锈的铁块堆积如山，散发着铁腥味
- 施罗德的表情:皱着眉头，厌恶，但又无能为力

## 第9章《谍战大沽口》

### 核心场景
老鬼的情报网开始行动，调查大沽口事件。

### 感官压抑元素（≥3处）
1. **铁币的交易**: 孙猴子（东厂探事）收老鬼的铁币作为情报费，铁币在两人之间传递，发出冰冷的碰撞声

陈铁把铁币倒在桌上。
